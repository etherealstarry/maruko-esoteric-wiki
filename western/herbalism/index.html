    草药学 · 学习笔记

    植物和灵性的连接，西方神秘学的重要分支。

    什么是草药学

    草药学（Herbalism / Hedge Witchery）是用植物的能量来疗愈、保护和占卜的学问。

    和中医草药学的区别： - 中医：看四气五味、归经，偏重"药性" - 西方：看植物灵（Plant Spirit）、元素对应、月相，偏重"灵性"

    核心想法： 植物不是"死的药材"，而是有意识的存在。你可以和植物"对话"。

    基础：植物和元素/行星的对应

    西方草药学把植物和占星体系对应起来。

    植物和行星对应（部分）

    行星 代表植物 用途
    ☉ 太阳 金盏花、玫瑰、肉桂 提升自信、领导力
    ☽ 月亮 茉莉、百合、柳树 情绪平衡、直觉开发
    ☿ 水星 薰衣草、薄荷、茴香 沟通、思维清晰
    ♀ 金星 玫瑰、草莓、苹果 爱情、美容、和谐
    ♂ 火星 辣椒、大蒜、生姜 保护、行动力、驱邪
    ♃ 木星 丁香、肉豆蔻、橡树 扩张、幸运、成长
    ♄ 土星 雪松、乌头（有毒！） 限制、 grounding、深层工作

    💡 记忆技巧：植物的"性格"和对应的行星一致。比如玫瑰（金星）= 爱情；辣椒（火星）= 保护。

    基础：植物和元素对应

    元素 代表植物 用途
    火 辣椒、肉桂、玫瑰mary 行动、保护、热情
    水 茉莉、百合、睡莲 情感、直觉、疗愈
    风 薰衣草、薄荷、尤加利 思维、沟通、净化
    土 雪松、橡树、姜黄 grounding、稳定、财富

    实用方法

    1. 草药茶（Tea Magic）

    最简单的入门方法。

    步骤： 1. 选对应你目标的草药 2. 烧水，等水开 3. 把草药放进去，盖上盖子（"锁住能量"） 4. 默念你的意图 5. 喝的时候，感受能量

    配方示例： - 爱情茶：玫瑰 + 肉桂 + 蜂蜜 - 保护茶：大蒜 + 生姜 + 柠檬 - 直觉茶：茉莉 + 薄荷 + 月见草

    2. 草药包（Herb Bundle）

    把干燥的草药包在布里，放在枕头下/口袋里/门口。

    用途： - 枕头下 = 助眠（薰衣草）、预知梦（月见草） - 口袋里 = 幸运（四叶草）、保护（大蒜） - 门口 = 保护家宅（迷迭香、鼠尾草）

    3. 烟熏（Smudging）

    用干燥的草药（通常是鼠尾草、雪松）燃烧，烟熏房间/人。

    用途： - 净化空间（去掉负能量） - 净化自己（仪式前/后） - 送走不好的东西

    ⚠️ 注意：烟熏是北美原住民的仪式，用的时候要尊重起源，不要"文化挪用"。

    4. 植物占卜（Botanical Divination）

    用植物的形态、生长情况来占卜。

    方法： - 茶叶占卜（Tasseography）：喝完茶，看茶叶在杯底的图案 - 树枝占卜（Radionics）：用树枝的弯曲来判断"是/否" - 植物生长：种的植物长得好 = 能量对；长不好 = 需要调整

    学习资源

    书籍

    入门： - 《The Green Witch》—— Arin Murphy-Hiscock，现代草药学，易懂 - 《Hedge Witchery》—— Raven Grimassi， hedge witch传统

    进阶： - 《Culpeper's Complete Herbal》—— Nicholas Culpeper，17世纪经典，植物和占星对应 - 《The Herbal Alchemist's Handbook》—— Karen Harrison，草药+魔法

    在线资源

    • Herb Riot —— 现代草药学
    • B站搜索"西方草药" —— 中文资源较少，但有一些

    购买

    • 淘宝搜索"干燥草药" —— 大部分都能买到
    • 本地花草茶店 —— 普通的也可以（薰衣草、玫瑰、迷迭香）

    学习步骤

    第1周：基础对应
  → 记植物的元素/行星对应
  → 从薰衣草、玫瑰、迷迭香开始（最容易买到）

第2-3周：实用方法
  → 学泡茶（最简单）
  → 每天泡一杯，感受能量

第4周：进阶
  → 学草药包、烟熏
  → 开始自己的"草药日记"

长期：
  → 读《The Green Witch》
  → 尝试植物占卜
  → 种自己的草药园（阳台也行）

    常见误区

    1. 误区：草药学就是"喝花草茶"
      ✅ 正确：花草茶只是入门。草药学包含能量工作、占卜、仪式。

    2. 误区：所有植物都能随便用
      ✅ 正确：有些植物有毒（乌头、颠茄），有些会过敏。用之前先查。

    3. 误区：需要"天赋"才能和植物对话
      ✅ 正确：植物一直在"说话"，只是你没注意。去阳台坐5分钟，什么都不做，感受植物，就有了。
