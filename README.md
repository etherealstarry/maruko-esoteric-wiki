## 这是什么

个人玄学学习笔记网站，记录从零开始学习西方神秘学和东方玄学的过程。

**两条主线：**
- 🔮 西方神秘学：塔罗、雷诺曼、占星、西方数术
- ☯ 东方玄学：八字、紫微、易经、奇门

## 本地运行

```bash
git clone https://github.com/etherealstarry/maruko-esoteric-wiki.git
cd maruko-esoteric-wiki
npm install
npm run dev
```

构建静态文件：
```bash
npm run build
```
