## 什么是奇门遁甲

奇门遁甲是中国古代最高级的术数，被称为**"帝王之学"**。传说黄帝战蚩尤时用奇门取胜。

**名字意思：**
- **奇**：乙、丙、丁三奇（三种吉的能量）
- **门**：开、休、生、伤、杜、景、死、惊八门
- **遁甲**：甲（最重要的天干）隐藏起来，由"值符"代行

**核心用途：**
- 决策：这件事该不该做？
- 择日：哪天做最好？
- 选方位：往哪个方向走？
- 风水：这个位置好不好？

**和其他术数的关系：**

| | 八字 | 紫微 | 奇门 |
|---|------|------|------|
| 用途 | 看一生运势 | 看性格细节 | 看具体决策/事件 |
| 时间 | 看大运（10年） | 看流年（1年） | 看时辰（2小时） |
| 精细度 | 粗 | 中 | 极细 |

## 核心要素：三奇 + 六仪 + 八门 + 九星 + 八神

### 三奇（吉）

| 奇 | 天干 | 五行 | 含义 |
|------|------|------|------|
| 日奇 | 乙 | 木 | 希望、柔顺、调和 |
| 月奇 | 丙 | 火 | 权威、光明、果断 |
| 星奇 | 丁 | 火 | 智慧、灵动、锐利 |

### 六仪

甲（戊己庚辛壬癸）天干，甲隐藏，由六仪代行。

### 八门（最实用）

| 门 | 吉凶 | 五行 | 含义 | 什么时候用 |
|------|------|------|------|------------|
| 开门 | **大吉** | 金 | 开始、公开、事业 | 开业、面试、公开活动 |
| 休门 | **大吉** | 水 | 休息、谈判、休养 | 谈判、休息、恢复 |
| 生门 | **大吉** | 土 | 生长、财富、生命 | 投资、种植、就医 |
| 伤门 | **凶** | 木 | 伤害、损失、争斗 | 尽量不用 |
| 杜门 | 平 | 木 | 隐藏、阻塞、技术 | 学习、闭关、技术工作 |
| 景门 | 平 | 火 | 文化、信息、美景 | 宣传、考试、发表 |
| 死门 | **大凶** | 土 | 死亡、终结、不动 | 葬礼、结束、绝不用 |
| 惊门 | **凶** | 金 | 惊恐、口舌、诉讼 | 尽量不用 |

> 💡 **实用记忆**：开休生 = 三吉门（尽量用）；伤死惊 = 三凶门（尽量避免）；杜景 = 中平。

## 解读方法（核心）

### 步骤1：定用神

用神 = 你问的事，在奇门盘里对应什么？

| 问什么 | 用神是什么 |
|--------|------------|
| 事业/工作 | 开门 |
| 感情/婚姻 | 六合 |
| 财运/生意 | 生门/戊 |
| 健康/疾病 | 天芮星 |
| 考试/面试 | 丁/天辅星 |
| 出行/旅行 | 伤门/白虎 |
| 打官司 | 惊门/白虎 |
| 找东西 | 太阴/死门 |

> 💡 **核心**：先定用神，再看用神落在哪个宫、宫里有什么、好不好。

### 步骤2：看用神落在哪个宫

奇门盘有9个宫（九宫八卦），用神落在哪个宫，就分析那个宫。

```
┌─────┬─────┬─────┐
│ 4巽 │ 9离 │ 2坤 │
│ 木  │ 火  │ 土  │
├─────┼─────┼─────┤
│ 3震 │ 5中 │ 7兑 │
│ 木  │ 土  │ 金  │
├─────┼─────┼─────┤
│ 8艮 │ 1坎 │ 6乾 │
│ 土  │ 水  │ 金  │
└─────┴─────┴─────┘
```

### 步骤3：分析那个宫

一个宫里同时有：门 + 星 + 神 + 天干

**解读框架：**

| 元素 | 问题 | 好坏 |
|------|------|------|
| 门 | 这件事好不好做？ | 吉门=好，凶门=难 |
| 星 | 事情的性质？ | 吉星=顺，凶星=不顺 |
| 神 | 有什么无形力量影响？ | 值符=贵人，白虎=凶 |
| 天干 | 五行生克，细节 | 相生=好，相克=不好 |

**例子：问考试，用神=丁（星奇）**
- 丁落在离宫，宫里有开门+天辅星+值符
- 解读：开门（吉门）+ 天辅（文昌星）+ 值符（贵人）= 考试顺利

## 学习经验

### 入门阶段（第1-2周）

**目标：** 记住八门的吉凶，记住三奇六仪。

**方法：**
1. 背八门口诀：开休生吉，杜景平，伤死惊凶
2. 背三奇：乙丙丁
3. 每天看一个"时辰盘"（用工具），找三吉门在哪里

**常见坑：**
- 不要一上来就学手工排盘（太复杂，会放弃）
- 不要硬背九星八神的五行（先知道名字和含义就行）
- 奇门是拿来用的，不是拿来背的

### 进阶阶段（第3-4周）

**目标：** 学会定用神，学会看"用神在哪个宫"。

**方法：**
1. 背用神对应表（问事业=开门，问感情=六合...）
2. 每天选一个实际问题（比如"今天适合谈合作吗？"），看当前时辰的奇门盘
3. 记录自己的判断和实际结果

### 精通阶段（第5-8周）

**目标：** 学会综合解读一个宫（门+星+神+天干），学会判断吉凶。

**方法：**
1. 学天干组合（乙+丙=日月同辉，好；庚+癸=冲突，不好）
2. 学旺衰（门在哪个宫最旺）
3. 开始给别人看（朋友做了重大决定，问这件事会不会好？）

## 记忆方法

### 方法1：口诀法

**八门：** 开休生吉杜景平，伤死惊凶记分明

**三奇：** 乙丙丁，日月星

**九星：** 蓬芮冲辅禽心柱任英

**八神：** 直符腾蛇太阴行，六合白虎玄武惊，九地九天到终点

### 方法2：用途记忆法

不硬背，用"什么情况用什么门"来记：

```
想开始一件事 → 用开门
想休息/谈判 → 用休门
想赚钱/投资 → 用生门
想学习/闭关 → 用杜门
想宣传/考试 → 用景门
绝不用 → 死门
```

### 方法3：每日一卦法

每天看一个时辰的奇门盘，问一个现实问题：
- 早上看：今天适合做什么？（查看三吉门的方向）
- 实际做：往三吉门的方向走/活动
- 晚上复盘：今天顺利吗？

**坚持30天，奇门的基础解读能力就建立起来了。**

## 重要提醒

1. 奇门是所有术数里最难的，不要想"一个月学会"
2. 先学八字和紫微（基础），再学奇门（进阶）
3. 奇门最实用的不是"算命"，而是**决策**——这件事该不该做？哪个方向最好？
4. 每天看一个时辰盘，坚持30天，比看书一个月有效
