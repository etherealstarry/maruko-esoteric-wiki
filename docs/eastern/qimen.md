# 奇门遁甲

> 高阶占卜术，用于决策和择时。

---

## 什么是奇门遁甲

奇门遁甲是中国古代的高阶占卜术，原本用于军事决策，后来发展出命理和风水应用。

**特点：**
- 体系复杂，需要长期学习
- 适合问"该怎么办"、"什么时候做"
- 有排盘（奇门盘），类似八字但更复杂

---

## 基础概念

### 奇门盘的结构

奇门盘分四层：
1. **地盘** —— 八卦、九宫
2. **天盘** —— 九星（天蓬、天芮等）
3. **人盘** —— 八门（开门、休门等）
4. **神盘** —— 八神（值符、腾蛇等）

### 八门

| 门 | 吉凶 | 代表 |
|----|------|------|
| 开门 | 吉 | 开始、公开 |
| 休门 | 吉 | 休息、谈判 |
| 生门 | 吉 | 生长、财运 |
| 伤门 | 凶 | 伤害、损失 |
| 杜门 | 平 | 阻塞、隐藏 |
| 景门 | 平 | 文书、信息 |
| 死门 | 凶 | 结束、固执 |
| 惊门 | 凶 | 惊恐、口舌 |

## 学习步骤

```
1. 学八卦、九宫（基础中的基础）
2. 学九星、八门、八神
3. 学排盘（用工具，不要手算）
4. 学怎么读盘（找用神）
5. 学实际应用
```

> ⚠️ **警告**：奇门是高阶内容，建议先学八字和易经。
