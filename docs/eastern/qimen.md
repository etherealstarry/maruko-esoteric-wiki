## 什么是奇门遁甲

奇门遁甲是中国古代最高级的术数之一，被称为**"帝王之学"**。

**为什么叫"奇门遁甲"？**
- **奇**：乙、丙、丁三奇（三种吉门）
- **门**：开、休、生、伤、杜、景、死、惊八门
- **遁甲**：甲（天干）隐藏起来，由"值符"代行

**核心用途：**
- 决策（这件事该不该做？）
- 择日（哪天做这件事最好？）
- 风水（这个方位好不好？）
- 预测（当前局势会怎么发展？）

## 基础概念

### 三奇（吉）

| 奇 | 天干 | 含义 |
|------|------|------|
| 日奇 | 乙 | 希望、柔顺 |
| 月奇 | 丙 | 权威、光明 |
| 星奇 | 丁 | 智慧、灵动 |

### 八门（最重要）

| 门 | 吉凶 | 含义 |
|------|------|------|
| 开门 | 大吉 | 开始、公开、事业 |
| 休门 | 大吉 | 休息、养精、谈判 |
| 生门 | 大吉 | 生长、财富、生命 |
| 伤门 | 凶 | 伤害、损失、争斗 |
| 杜门 | 平 | 隐藏、阻塞、技术 |
| 景门 | 平 | 文化、信息、美景 |
| 死门 | 大凶 | 死亡、终结、不动 |
| 惊门 | 凶 | 惊恐、口舌、诉讼 |

> 💡 **记忆技巧**：开门、休门、生门 = 三吉门，优先用。伤门、死门、惊门 = 三凶门，尽量避。

## 排盘（超级复杂，用工具！）

奇门遁甲的排盘是**所有术数里最复杂的**，手工排盘需要：
1. 定局（阳遁/阴遁，几局？）
2. 排地盘（九宫八卦）
3. 排天盘（九星飞布）
4. 排人盘（八门）
5. 排神盘（八神）

**不要手算！** 用排盘工具。

### 推荐排盘工具

- **[奇门遁甲排盘](https://www.qimen.org/)** —— 在线，免费
- **App搜索"奇门遁甲"** —— iOS/Android都有
- **[文墨天机](https://www.wenmotianji.com/)** —— 也有奇门排盘

## 如何解读

### 步骤1：看"用神"

用神 = 你问的事对应的符号。

**例子：**
- 问事业 → 用神 = 开门（事业门）
- 问财运 → 用神 = 生门（财门）
- 问感情 → 用神 = 六合（婚姻神）
- 问健康 → 用神 = 天芮星（病星）

### 步骤2：看用神落在哪个宫

奇门盘有9个宫（九宫八卦），用神落在哪个宫，就分析那个宫。

**九宫八卦：**

```
┌─────┬─────┬─────┐
│  4  │  9  │  2  │
│ 巽  │  离  │  坤  │
├─────┼─────┼─────┤
│  3  │  5  │  7  │
│ 震  │  中  │  兑  │
├─────┼─────┼─────┤
│  8  │  1  │  6  │
│ 艮  │  坎  │  乾  │
└─────┴─────┴─────┘
```

### 步骤3：分析那个宫里的信息

一个宫里有：
- 门（八门之一）
- 星（九星之一）
- 神（八神之一）
- 天干（三奇六仪）

综合分析，得出结论。

## 学习建议

**奇门遁甲是高级术数，建议在学完以下内容后再学：**
1. 八字（基础）
2. 紫微斗数（精细）
3. 易经（哲学基础）

**为什么？**
- 奇门太复杂，没有基础直接学会很痛苦
- 八字和紫微学会了，学奇门会快很多
- 奇门需要"术数感"，先学简单的培养感觉

## 常见误区

1. **误区：奇门可以"改命"**  
   ✅ 正确：奇门是决策工具，帮你选最好的时间和方向，但不能改变命运。

2. **误区：一定要会手工排盘**  
   ✅ 正确：现代有排盘工具，手工排盘是"了解原理"，不是"必须掌握"。

3. **误区：奇门是"迷信"**  
   ✅ 正确：奇门是一套复杂的时空编码系统，和现代的系统论、控制论有相通之处。
