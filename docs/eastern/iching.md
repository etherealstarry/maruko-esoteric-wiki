## 什么是易经

《易经》是中国最古老的经典之一，原本是占卜书，后来发展出哲学体系。

**易经占卜**是用50根蓍草或三枚硬币，得到一个**卦**，然后根据卦象来判断吉凶、做决策。

**核心思想：**
- 变易（变化是唯一的不变）
- 简易（复杂事物有简单规律）
- 不易（变化的规律是不变的）

## 基础概念

### 八卦（8个基本符号）

八卦是易经的基础符号，每个卦有3爻（yao）：

```
☰ 乾（天）   ☱ 兑（泽）   ☲ 离（火）   ☳ 震（雷）
☴ 巽（风）   ☵ 坎（水）   ☶ 艮（山）   ☷ 坤（地）
```

| 卦 | 名 | 象征 | 关键词 |
|------|------|----------|--------|
| ☰ | 乾 | 天、父 | 健、创造、主动 |
| ☷ | 坤 | 地、母 | 顺、接受、被动 |
| ☳ | 震 | 雷 | 动、启动、惊醒 |
| ☵ | 坎 | 水 | 陷、危险、智慧 |
| ☲ | 离 | 火 | 丽、明亮、依附 |
| ☶ | 艮 | 山 | 止、静止、稳定 |
| ☴ | 巽 | 风 | 入、渗透、柔顺 |
| ☱ | 兑 | 泽 | 悦、快乐、口舌 |

> 💡 **记忆技巧**：乾三连（☰），坤六断（☷），震仰盂（☳），艮覆碗（☶），离中虚（☲），坎中满（☵），兑上缺（☱），巽下断（☴）。

### 六十四卦

两个八卦重叠，得到64卦（每个卦有6爻）。

**重要卦象：**

| 卦 | 名 | 含义 |
|------|------|------|
| ䷀ | 乾为天 | 纯阳，刚健，创造 |
| ䷁ | 坤为地 | 纯阴，柔顺，接受 |
| ䷂ | 水雷屯 | 开始，困难，萌芽 |
| ䷃ | 山水蒙 | 启蒙，学习，无知 |
| ... | ... | ... |
| ䷾ | 水火既济 | 事情完成，平衡 |
| ䷿ | 火水未济 | 事情未完成，需要继续努力 |

> 💡 **学习技巧**：先学8个八卦，再学64卦的"上卦+下卦"组合逻辑，不需要死背64个卦名。

## 占卜方法

### 方法一：硬币法（最简单，推荐入门）

1. 准备三枚硬币（一面为"正"，一面为"反"）
2. 默念问题（要具体）
3. 抛硬币六次，从下往上记录：
   - **两正一反** = 少阳（—，阳爻）
   - **两反一正** = 少阴（--，阴爻）
   - **三正** = 老阳（—o，阳爻，但会变）
   - **三反** = 老阴（--x，阴爻，但会变）
4. 从下往上记录，得到**本卦**
5. 如果有"老阳"或"老阴"（变爻），变爻阴阳互换，得到**之卦**
6. 查《易经》，看本卦和之卦的卦辞、爻辞

> 💡 **变爻的判断**：
> - 没有变爻 → 看本卦卦辞
> - 一个变爻 → 看本卦那个爻的爻辞
> - 两个变爻 → 看本卦那两个爻的爻辞（以下爻为主）
> - 三个变爻 → 看本卦卦辞 + 之卦卦辞
> - 四个及以上变爻 → 看之卦卦辞

### 方法二：蓍草法（传统，复杂）

用50根蓍草，经过分、挂、揲、归等步骤，得到6爻。

**优缺点：**
- ✅ 最传统，最有"仪式感"
- ❌ 太复杂，初学者不推荐

### 方法三：数字法（最快）

1. 随意想两个数字（1-999）
2. 第一个数字 ÷ 8，取余数（余数0=8），得上卦
3. 第二个数字 ÷ 8，取余数，得下卦
4. 两个数字相加 ÷ 6，取余数（余数0=6），得变爻
5. 得到卦象和变爻，查《易经》

**例子：**
- 想的数字：23 和 45
- 23 ÷ 8 = 2余7 → 上卦 = 7 = 艮（山）
- 45 ÷ 8 = 5余5 → 下卦 = 5 = 坎（水）
- (23+45) ÷ 6 = 11余2 → 变爻 = 第2爻
- 得到：**水山蹇（䷦）**，第2爻变 → 之卦 = **水地比（䷄）**

## 如何解读

### 步骤1：看卦名和卦辞

卦名和卦辞是"总纲"，告诉你这个卦的整体能量。

**例子：水火既济（䷾）**
- 卦名：既济 = 已经渡过河 = 完成
- 卦辞：事已成，但要防微杜渐
- 整体：事情完成了，但要小心，不要大意

### 步骤2：看变爻（如果有）

变爻是"当前最需要注意的地方"。

**例子：乾卦（䷀），初九爻变**
- 初九爻辞："潜龙勿用"
- 意思：龙还在潜伏，不要轻举妄动
- 建议：现在不是行动的时候，要等待

### 步骤3：结合问题背景

同一个卦，在不同问题里意思不同。

**乾为天（䷀）的例子：**
- 问事业 → 适合开创，有领导力，但要小心骄傲
- 问感情 → 对方很强，但可能太自我
- 问健康 → 身体好，但要注意头部、脊椎

## 卦象的记忆方法

### 方法1：上下卦组合法

64卦 = 上卦（上面那个八卦）+ 下卦（下面那个八卦）

**例子：**
- **水火既济** = 上卦水（坎）+ 下卦火（离）
- **火水未济** = 上卦火（离）+ 下卦水（坎）

> 💡 **记忆技巧**：先记8个八卦，再记"上+下"的组合，64卦就全理解了，不需要死背。

### 方法2：卦序记忆法

《易经》有固定的卦序（第1卦乾，第2卦坤，第3卦屯...），可以按顺序记。

**前8卦：**
1. 乾为天
2. 坤为地
3. 水雷屯
4. 山水蒙
5. 水天需
6. 天水讼
7. 地水师
8. 水地比

> 💡 **记忆技巧**：卦序有逻辑，不是随机排的。比如"屯"（开始困难）之后是"蒙"（需要启蒙），有因果关系。

### 方法3：关键词法

给每个卦记1-2个关键词。

**例子：**
- 乾 = 创造、主动
- 坤 = 接受、被动
- 屯 = 开始、困难
- 蒙 = 学习、启蒙
- 既济 = 完成
- 未济 = 未完成

## 常见误区

1. **误区：易经是迷信**  
   ✅ 正确：《易经》是中国最古老的智慧，包含哲学、决策学、系统思维。孔子说"加我数年，五十以学易，可以无大过矣"。

2. **误区：占卜就是算命**  
   ✅ 正确：易经占卜是**决策工具**，不是算命。它帮你看到当前局势和可能的结果，但决定还是你自己做。

3. **误区：一定要用蓍草**  
   ✅ 正确：硬币法、数字法都可以，重要的是你的诚心和直觉，不是工具。

4. **误区：卦辞爻辞要背得一字不差**  
   ✅ 正确：理解核心意思，结合问题背景灵活解读。
