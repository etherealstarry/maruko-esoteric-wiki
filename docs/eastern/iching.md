# 易经占卜

> 用卦象来决策和预测，适合问"该怎么办"。

---

## 什么是易经占卜

《易经》是中国最古老的经典之一，原本是占卜书，后来发展出哲学体系。

**易经占卜**是用50根蓍草或三枚硬币，得到一个**卦**，然后根据卦象来判断吉凶。

---

## 基础概念

### 八卦

八卦是易经的基础符号：

```
☰ 乾（天）   ☱ 兑（泽）   ☲ 离（火）   ☳ 震（雷）
☴ 巽（风）   ☵ 坎（水）   ☶ 艮（山）   ☷ 坤（地）
```

### 六十四卦

两个八卦重叠，得到64卦（每个卦有6爻）。

例如：
- **乾为天**（䷀）—— 纯阳，刚健
- **坤为地**（䷁）—— 纯阴，柔顺
- **水火既济**（䷾）—— 事情完成
- **火水未济**（䷿）—— 事情未完成

## 占卜方法

### 硬币法（最简单）

1. 准备三枚硬币
2. 默念问题
3. 抛硬币六次，记录结果：
   - 两正一反 = 少阳（—）
   - 两反一正 = 少阴（--）
   - 三正 = 老阳（—o，变爻）
   - 三反 = 老阴（--x，变爻）
4. 从下往上记录，得到本卦
5. 有变爻的话，变爻阴阳互换，得到之卦
6. 查《易经》，看卦辞、爻辞

## 学习方法

```
1. 先学八卦象征（乾=天=父=健...）
2. 学六十四卦名和基本概念
3. 学硬币占卜法
4. 学怎么查卦辞、爻辞
5. 学怎么解读
```
