> "相"是东方玄学五大体系之一，风水是"相地"——看环境和人的关系。

## 什么是风水

风水（Feng Shui）是通过调整**环境布局**来改变气场、改善运势的方法。

**字面意思：**
- 风 = 气流（能量流动）
- 水 = 水流（财富流动）

**核心思想：**
- 气（能量）会流动，环境会影响气的流动
- 调整环境，就能调整气的流动
- 气的流动好了，人的状态就会好

## 两大流派

| 流派 | 特点 | 适合 |
|--------|--------|--------|
| **形势派** | 看山看水，大环境 | 选房子、选墓地 |
| **理气派** | 看方位、理气，室内布局 | 室内风水、办公风水 |

> 💡 **初学者建议**：先学理气派（实用，能马上用），形势派后学。

## 基础：八卦和方位

风水用**八卦**来对应方位：

```
        ☰ 乾（西北）= 父 = 事业/权力
    ☱ 兑（西）= 少女 = 口舌/喜悦
☴ 巽（东南）= 长女 = 财富/文书
    ☲ 离（南）= 中女 = 名声/光明
☵ 坎（北）= 中男 = 事业/险难
    ☶ 艮（东北）= 少男 = 稳定/学习
☳ 震（东）= 长男 = 行动/健康
    ☷ 坤（西南）= 母 = 婚姻/家庭
```

> 💡 **记忆技巧**：站在中国（北半球），南=热=火=离；北=冷=水=坎。

## 实用风水（家里马上能用）

### 1. 大门（最重要）

- 大门是"气口"，所有气从大门进
- **不要**正对厕所门（财气被冲）
- **不要**正对楼梯（气直冲）
- **化解**：放屏风、门帘、或者在大门放一盆阔叶绿植

### 2. 客厅

- 客厅在**房子前半部**最好（气进来先到客厅）
- 沙发要靠实墙（"靠山"）
- 不要梁压沙发（坐下面有压迫感）
- **财位**：大门对角线的角落，放绿植或聚宝盆

### 3. 卧室

- 床头要靠实墙（不能靠窗、不能靠厕所墙）
- 镜子不能对床（晚上起来会被自己吓到，影响睡眠）
- 卧室不要放太多电器（辐射 + 火气太旺）
- 颜色温和（不要大红大紫，影响睡眠）

### 4. 厨房

- 厨房属"火"，不要和厕所相邻（水火相冲）
- 炉灶不要正对水槽（水火相冲）
- **化解**：中间放一块木板/绿植隔断

### 5. 书房/办公桌

- 办公桌最好**坐实向虚**（背后是实墙，前面是空地/窗）
- 不要背对门（背后有人走过，不专注）
- 左手边（青龙位）放高一点的东西（文件架、植物），右手边（白虎位）放低一点的东西

## 化煞方法

| 煞 | 是什么 | 化解 |
|--------|--------|--------|
| 门冲 | 两个门正对 | 门帘、屏风 |
| 路冲 | 马路直冲房子 | 门口放石敢当、种树 |
| 尖角煞 | 对面楼的墙角正对 | 窗口放凸面镜 |
| 天斩煞 | 两栋楼中间缝隙正对 | 种树、放屏风 |
| 横梁压顶 | 梁在头顶 | 吊顶、挂葫芦 |

> ⚠️ **注意**：凸面镜是"反射"煞气，不能随便挂——会对着邻居，引起纠纷。最好用植物化解。

```
第1周：基础概念
  → 学八卦对应方位

第2-3周：实用风水
  → 学家里各个区域的风水
  → 调整自己的房间/家
  → 记录变化

第4周：化煞
  → 学常见煞的化解方法
  → 判断自己家有没有煞

长期：
  → 读《八宅明镜》
  → 学形势派（选房、选地）
  → 持续调整，记录效果
```

## 常见误区

1. **误区：风水是迷信**  
   ✅ 正确：风水有环境心理学基础——采光、通风、布局，确实影响人的状态。

2. **误区：一定要请风水师**  
   ✅ 正确：基础风水（上面这些）自学就能会，自己调整就好。

3. **误区：风水能"逆天改命"**  
   ✅ 正确：风水是"助力"，不是"决定"。八字该有的，风水帮你发挥；八字没有的，风水帮不了太多。
