# 中医基础 · 医

> "医"是东方玄学五大体系之一，用阴阳五行来理解身体和疾病。

---

## 什么是"医"

"医"是道家五术里的**医学体系**，核心是**中医**。

**和现代西医的区别：**
- 西医：看"病菌""指标"，治标
- 中医：看"阴阳""气血"，治本

**核心思想：**
- 人体是一个整体（不是零件拼起来的）
- 人和自然是一个整体（天人合一）
- 疾病是"不平衡"，治疗是"恢复平衡"

## 气血津液

### 气（Qi）

气是生命的**能量/动力**。

| 气 | 作用 |
|--------|--------|
| 元气 | 先天之本，肾里 |
| 宗气 | 呼吸之气，胸中 |
| 营气 | 营养之气，血管里 |
| 卫气 | 保卫之气，体表 |

### 血（Blood）

血是**滋养物质**。

- 心主血（心脏泵血）
- 肝藏血（肝储存血）
- 脾统血（脾控制血不溢出）

### 津液（Body Fluids）

- 津 = 清稀的液体（汗、泪、唾液）
- 液 = 稠厚的液体（髓、精）

## 经络基础

经络是**气血运行的通道**。

### 十二正经

| 经络 | 走向 | 对应脏腑 |
|--------|--------|----------|
| 手太阴肺经 | 胸→手 | 肺 |
| 手阳明大肠经 | 手→头 | 大肠 |
| 足阳明胃经 | 头→脚 | 胃 |
| 足太阴脾经 | 脚→胸 | 脾 |
| 手少阴心经 | 胸→手 | 心 |
| 手太阳小肠经 | 手→头 | 小肠 |
| 足太阳膀胱经 | 头→脚 | 膀胱 |
| 足少阴肾经 | 脚→胸 | 肾 |
| 手厥阴心包经 | 胸→手 | 心包 |
| 手少阳三焦经 | 手→头 | 三焦 |
| 足少阳胆经 | 头→脚 | 胆 |
| 足厥阴肝经 | 脚→胸 | 肝 |

> 💡 **记忆技巧**：经络走向有规律——手三阴从胸走手，手三阳从手走头；足三阳从头走脚，足三阴从脚走胸。

## 四诊（诊断方法）

### 1. 望诊（看）

- **看面色**：青=肝，红=心，黄=脾，白=肺，黑=肾
- **看舌**：舌质（颜色）+ 舌苔（厚薄、颜色）
- **看形态**：胖瘦、精神

### 2. 闻诊（听/闻）

- **听声音**：咳嗽、说话声音（响亮=实证，低微=虚证）
- **闻气味**：口气、体味

### 3. 问诊（问）

- **问寒热**：怕冷还是怕热？
- **问汗**：出汗多不多？
- **问疼痛**：哪里痛？怎么个痛法？
- **问二便**：大小便正常吗？
- **问饮食**：胃口好吗？口渴吗？

### 4. 切诊（摸）

- **把脉**：寸关尺三部，浮沉迟数...（最难，需要长期学）
- **按腹**：肚子软硬？有没有包块？

## 八纲辨证（诊断框架）

所有疾病，都可以用**八纲**来分类：

| 纲 | 含义 |
|--------|--------|
| 阴阳 | 总纲（病是阴证还是阳证？） |
| 表里 | 病在哪里？（体表还是内脏？） |
| 寒热 | 病的性质（怕冷还是怕热？） |
| 虚实 | 病的态势（正气足还是不足？） |

> 💡 **核心**：先分阴阳，再看表里寒热虚实，治疗方向就清楚了。

## 常见养生方法

### 1. 食疗（最安全）

```
五味入五脏：
酸入肝，苦入心，甘入脾，辛入肺，咸入肾
```

| 食材 | 性味 | 功效 |
|--------|--------|--------|
| 红枣 | 甘温 | 补脾、养血 |
| 生姜 | 辛温 | 散寒、暖胃 |
| 绿豆 | 甘寒 | 清热、解毒 |
| 黑豆 | 甘平 | 补肾、利水 |
| 百合 | 甘微寒 | 润肺、安神 |

### 2. 艾灸（温阳）

- 用艾草燃烧，灸穴位
- 最适合：怕冷、阳虚的人
- 常用穴位：足三里（强身）、关元（补元气）、三阴交（调妇科）

### 3. 拔罐（祛湿）

- 用罐子吸住皮肤，造成局部淤血
- 作用：祛湿、活血、散寒
- 最适合：肌肉酸痛、湿气重

### 4. 刮痧（疏通经络）

- 用刮痧板刮皮肤，出"痧"（红紫印）
- 作用：疏通经络、排毒
- 最适合：感冒初期、肩颈酸痛

## 常见误区

1. **误区：中医是"慢郎中"**  
   ✅ 正确：急性病（感冒、肠胃炎）中医也很快；慢性病（调理体质）需要时间。

2. **误区：中药无毒副作用**  
   ✅ 正确：中药也有毒副作用，不能乱吃。"是药三分毒"。

3. **误区：每个人都要"进补"**  
   ✅ 正确：虚才补，不虚补了反而生病。先辨体质，再养生。

4. **误区：中医不科学**  
   ✅ 正确：中医的"科学"和西医的"科学"是两套体系。现代医学也在研究经络、针灸（已有证据证明有效）。
