# 面相 · 完全学习笔记

> "相由心生"——你的内在状态，会写在你脸上。

---

## 什么是面相

面相是通过观察**面部特征**来推断性格、健康、运势的方法。

**核心原理：**
- 长期情绪会影响面部肌肉（常皱眉→眉间纹；常笑→鱼尾纹上扬）
- 面相不是"命运"而是"当前状态的反映"
- "心改则相改"——内在变了，相也会变

## 十二宫（面部区域）

| 宫位 | 位置 | 代表 |
|------|------|------|
| 命宫 | 两眉之间（印堂） | 性格、命运基调 |
| 财帛宫 | 鼻头 | 财富、理财能力 |
| 兄弟宫 | 眉毛 | 兄弟姐妹、朋友 |
| 夫妻宫 | 眼尾 | 婚姻、感情 |
| 子女宫 | 眼下 | 子女、晚辈 |
| 疾厄宫 | 鼻根 | 健康、抵抗力 |
| 迁移宫 | 额头两侧 | 外出运 |
| 官禄宫 | 额头正中 | 事业、社会地位 |
| 田宅宫 | 眉毛上方 | 房产、家庭 |
| 福德宫 | 额头上方 | 福气、心态 |
| 父母宫 | 额角 | 父母、长辈 |

> 💡 **记忆**：中轴线（命宫→财帛宫）最重要；两侧对称。

## 五官详解

### 1. 眉

| 眉型 | 性格 | 注意事项 |
|------|------|----------|
| 浓眉 | 刚强、行动力 | 可能霸道 |
| 淡眉 | 温和、内向 | 可能没主见 |
| 眉尾散 | 钱存不住 | 学会记账 |
| 眉尾聚 | 会理财 | 好 |
| 眉压眼 | 压抑、压力大 | 需要释放 |
| 断眉 | 性格突变 | 注意情绪管理 |

### 2. 眼

| 眼型 | 性格 |
|------|------|
| 大眼 | 开朗、感性 |
| 小眼 | 内敛、理性 |
| 眼尾上 | 有野心、性格强 |
| 眼尾下 | 温和、易心软 |
| 三白眼 | 性格极端 |

### 3. 鼻

| 鼻型 | 含义 |
|------|------|
| 鼻头圆 | 财运好 |
| 鼻头尖 | 精明、挑剔 |
| 鼻梁高 | 自尊强 |
| 鼻梁低 | 温和、依赖 |

> 💡 **鼻=财运+自尊**。鼻头看钱，鼻梁看自尊。

### 4. 嘴 5. 耳

- 唇厚=重感情；唇薄=理性
- 嘴角上=乐观；嘴角下=悲观
- 耳大垂厚=福气好

## 气色（比形状重要！）

| 面色 | 含义 |
|------|------|
| 红润 | 健康、好运 |
| 青/暗 | 疲劳、压力 |
| 黄 | 脾胃不好 |
| 白 | 气血不足 |

> 💡 **实用**：早上照镜子看气色——比算命准。

## 学习方法

### 入门（1周）
1. 记住十二宫位置
2. 看自己的脸，对应各宫
3. 照镜子看气色

### 进阶（2周）
1. 看朋友的五官
2. 验证你的判断（问朋友对不对）
3. 开始"面相日记"

### 记忆口诀
```
印堂命宫眼尾妻
鼻头财帛眉兄弟
额中事业额上福
气色第一形状次
```
