# 面相 · 学习笔记

> "相"是东方玄学五大体系之一，看外表来推断内在。

---

## 什么是面相

面相（Face Reading / Physiognomy）是通过观察人的**面部特征**来推断性格、命运、健康的方法。

**核心思想：**
- "相由心生" —— 内在状态会影响外表
- 面部不同区域对应不同人生领域
- 不是"定数"，而是"当前状态的反映"

---

## 十二宫（面部区域划分）

面相把脸分成12个区域，对应12个人生领域：

| 宫位 | 位置 | 代表 |
|--------|--------|--------|
| 命宫（印堂） | 两眉之间 | 性格、命运基调 |
| 财帛宫 | 鼻头 | 财富、收入 |
| 兄弟宫 | 眉毛 | 兄弟姐妹、朋友 |
| 夫妻宫 | 眼尾（奸门） | 婚姻、感情 |
| 子女宫 | 眼下（泪堂） | 子女、晚辈 |
| 疾厄宫 | 山根（鼻梁根部） | 健康、疾病 |
| 迁移宫 | 额头两侧 | 外出、搬家、远方 |
| 奴仆宫 | 下巴两侧 | 下属、晚辈 |
| 官禄宫（事业宫） | 额头正中 | 事业、社会形象 |
| 田宅宫 | 眉毛上方 | 房产、家庭 |
| 福德宫 | 额头上方 | 福气、精神 |
| 父母宫 | 日角月角（额头两侧） | 父母、长辈 |

> 💡 **记忆技巧**：从中间开始——命宫（眉心）→ 往左右 → 往下（鼻、下巴）→ 往上（额头）。

## 五官解读

### 1. 眉毛

| 眉型 | 性格 |
|--------|--------|
| 浓眉 | 性格刚强、行动力强 |
| 淡眉 | 性格温和、内向 |
| 眉尾散 | 财运不稳、存不住钱 |
| 眉尾聚 | 财运稳、会理财 |
| 眉压眼 | 压力大、性格压抑 |

### 2. 眼睛

| 眼型 | 性格 |
|--------|--------|
| 大眼 | 开朗、感性、表达强 |
| 小眼 | 内敛、理性、深思 |
| 眼尾上扬 | 性格强、有野心 |
| 眼尾下垂 | 温和、善良、易心软 |
| 三白眼/四白眼 | 性格极端、自私（少见） |

### 3. 鼻子

| 鼻型 | 含义 |
|--------|--------|
| 鼻头圆润 | 财运好、会理财 |
| 鼻头尖 | 精明、挑剔、财运不稳 |
| 鼻梁高 | 自尊心强、有主见 |
| 鼻梁低 | 温和、依赖心强 |
| 鼻翼饱满 | 存钱能力强 |

> 💡 **核心**：鼻子看财运（鼻头）+ 自尊（鼻梁）。

### 4. 嘴巴

| 嘴型 | 含义 |
|--------|--------|
| 嘴大 | 开朗、能说、社交强 |
| 嘴小 | 内敛、谨慎、细腻 |
| 唇厚 | 感性、重感情 |
| 唇薄 | 理性、冷静、口才好 |
| 嘴角上扬 | 乐观、人缘好 |
| 嘴角下垂 | 悲观、易抱怨 |

### 5. 耳朵

| 耳型 | 含义 |
|--------|--------|
| 耳大垂厚 | 福气好、长寿 |
| 耳小贴脑 | 聪明、内敛 |
| 耳垂饱满 | 晚年运好、有福 |
| 耳轮反（招风耳） | 反叛、独立 |

## 气色（面色）

面相不只是"形状"，还有"气色"。

| 面色 | 含义 |
|--------|--------|
| 红润 | 健康、好运 |
| 青/黑 | 疲劳、肾不好、压力大 |
| 黄/暗 | 脾胃不好、湿气重 |
| 白/无血色 | 气血不足、怕冷 |

> 💡 **实用**：早上照镜子，看自己的气色——比算命还准。

## 学习步骤

```
1. 先学十二宫的位置（对应人生领域）
2. 再学五官解读（眉眼鼻嘴耳）
3. 学气色（面色反映健康/运势）
4. 实战：看自己的脸，对照；看朋友，验证
```

## 常见误区

1. **误区：面相是"迷信"**  
   ✅ 正确：面相有心理学基础——长期情绪会影响面部肌肉（比如常皱眉的人，眉间会有纹路）。

2. **误区：面相是"定数"**  
   ✅ 正确："相由心生"，内在变了，相也会变。古人说"心改则相改"。

3. **误区：一定要"老法师"才能看**  
   ✅ 正确：基础的面相（十二宫、五官），自学就能会。
