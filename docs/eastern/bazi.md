# 八字 · 完全学习笔记

> 东方玄学的基础。学会排盘和读干支，其他术数就好学了。

---

## 什么是八字

**八字**（又称四柱命理）是用一个人的出生时间（年/月/日/时）来推算命运趋势的方法。

为什么叫"八字"？
- 年、月、日、时 = **四柱**
- 每一柱有一个天干 + 一个地支 = **两字**
- 四柱 × 两字 = **八字**

例子：
```
年柱    月柱    日柱    时柱
甲子    丙寅    戊辰    庚申
```

## 第二步：学会排盘

**不要手算！** 用排盘工具。

### 推荐排盘工具

- **[文墨天机](https://www.wenmotianji.com/)** —— 专业八字排盘，免费，最推荐
- **[八字排盘App](https://apps.apple.com/cn/app/八字排盘)** —— iOS/Android都有
- **[紫微斗数排盘](https://www.freezd.com/)** —— 同时排八字和紫微

### 排盘需要什么信息？

1. **公历出生日期**（年/月/日）
2. **出生时间**（精确到小时，最好知道分钟）
3. **出生地点**（用来算真太阳时）

> ⚠️ **重要**：时柱很关键，如果出生时间不准，时柱可能排错，整个八字就偏了。

## 第三步：读懂四柱

排出来的盘，你要能看懂每根柱子是什么意思。

### 年柱 = 祖辈/童年/1-16岁

- 代表你的家族背景、祖辈情况
- 也看你小时候的成长环境
- 年干看祖父，年支看祖母

### 月柱 = 父母/青年/17-32岁

- 代表你的父母、家庭关系
- 也看你青年时期的运势
- 月干看父亲，月支看母亲（现代也有用年干看父亲的）

### 日柱 = 自己/中年/33-48岁

- **日干 = 你自己**（称为"日主"或"身"）
- **日支 = 你的配偶宫**（婚姻状况）

> 💡 **核心**：日干是你，所有其他干支都是"和你的关系"。

### 时柱 = 子女/晚年/49岁后

- 代表你的子女、晚年运势
- 时也看你的内心世界、潜意识

## 第四步：五行生克

五行是八字的核心分析工具。

### 五行相生（好的循环）

```
木 → 火 → 土 → 金 → 水 → 木
```

- 木生火（木头可以烧火）
- 火生土（火烧完变成灰土）
- 土生金（土里挖出金属）
- 金生水（金属熔化变成液态）
- 水生木（水浇树，树生长）

### 五行相克（制约/冲突）

```
木 → 土 → 水 → 火 → 金 → 木
```

- 木克土（树根破土）
- 土克水（土可以筑坝挡水）
- 水克火（水可以灭火）
- 火克金（火可以熔化金属）
- 金克木（金属可以砍树）

> 💡 **记忆技巧**：相生 = 母亲生孩子（付出）；相克 = 管理者制约下属（控制）。

## 第五步：十神（核心中的核心）

**十神**是八字最重要、也最难的概念。它是用"日干"和其他干支的关系来定义的。

### 十神表（以日干为"我"）

假设日干是**甲木**（你是甲木），其他干支和你的关系：

| 五行关系 | 同性（偏） | 异性（正） |
|----------|------------|------------|
| 生我（妈妈） | 偏印（枭神） | 正印 |
| 我生（子女） | 伤官 | 食神 |
| 克我（官杀） | 七杀（偏官） | 正官 |
| 我克（财） | 偏财 | 正财 |
| 同我（兄弟） | 比肩 | 劫财 |

> 💡 **记忆技巧**：
> - "正" = 阴阳不同（异性），"偏" = 阴阳相同（同性）
> - 生我 = 印（妈妈生我），我生 = 食伤（我生孩子）
> - 克我 = 官杀（管我的人），我克 = 财（我追求的东西）

### 十神代表什么（详细版）

| 十神 | 正面含义 | 负面含义 |
|------|----------|----------|
| **正印** | 母亲、学业、贵人、稳定、包容 | 依赖、被动、缺乏行动力 |
| **偏印（枭神）** | 继母、冷门学问、直觉、灵感 | 孤僻、冷漠、不切实际 |
| **食神** | 才华、口福、温和、享受 | 懒散、放纵、缺乏进取 |
| **伤官** | 才华、叛逆、锋芒、艺术 | 傲慢、冲动、口舌是非 |
| **正官** | 事业、丈夫、规矩、责任 | 压力、约束、保守 |
| **七杀（偏官）** | 压力、野心、危机、权威 | 暴力、极端、不安定 |
| **正财** | 稳定收入、妻子、务实 | 吝啬、物质主义 |
| **偏财** | 意外之财、情人、社交 | 不稳定、风流、投机 |
| **比肩** | 朋友、竞争、自力更生 | 固执、竞争、破财 |
| **劫财** | 朋友、行动力、慷慨 | 破财、争斗、冲动 |

## 第六步：看日主强弱

**日主强弱** = 你（日干）在八字里是强还是弱。

### 什么影响日主强弱？

1. **得令**（最重要）：出生月份对你的五行有利？
   - 比如你是甲木，生在春天（寅卯月）= 得令（强）
   - 生在秋天（申酉月）= 失令（弱）

2. **得地**：地支里有你的根（同类五行）？
   - 甲木的根 = 寅、卯（木的地支）
   - 地支有寅或卯 = 得地（有根，强）

3. **得势**：天干有其他同类帮你？
   - 天干有甲或乙 = 得势（有帮手，强）

### 为什么日主强弱重要？

- **身强**：能担财官（有能力承受好运）
- **身弱**：担不住（好运来了也容易错过，需要印比帮身）

> 💡 **简单判断**：日主在八字里得到的生助多 = 身强；克泄多 = 身弱。

---

## 第七步：找喜用神（最关键的一步）

**喜用神** = 对你最有利的五行。

- 身强 → 喜克泄耗（官杀、食伤、财）
- 身弱 → 喜生扶（印、比劫）

**为什么找喜用神？**
- 知道喜用神，就能知道：
  - 适合什么行业？（喜木火 → 教育、文化、互联网）
  - 适合什么方向？（喜木 → 东方；喜火 → 南方）
  - 适合和什么样的人在一起？（喜金 → 找金旺的人）

## 第八步：大运和流年（时间维度）

八字不是静态的，而是**随着时间变化**的。

### 大运

- 大运 = 10年一个阶段，看这10年的整体趋势
- 大运从几岁开始？看性别和出生年份的阴阳
- 大运的五行 = 这10年你"经历"的五行

### 流年

- 流年 = 每一年，看这一年的具体情况
- 比如2026年是丙午年（火很旺）
- 如果你的喜用神是火 → 2026年好运
- 如果你的忌神是火 → 2026年要小心

### 怎么看流年？

```
流年干支 + 大运干支 + 本命八字 = 今年的运势
```

三者互动，不是孤立看的。

## 常见误区

1. **误区：八字能算出一切**  
   ✅ 正确：八字看趋势，不是定数。努力可以改变很多。古人说"命自我立，福自己求"。

2. **误区：身强就是好，身弱就是不好**  
   ✅ 正确：身强身弱都要看喜用神。身强如果忌神多，也不一定好；身弱如果有印比帮身，也可以很好。

3. **误区：十神意思要背得一字不差**  
   ✅ 正确：理解逻辑，组合时灵活解读（和雷诺曼有点像！）。

4. **误区：八字是迷信**  
   ✅ 正确：八字是一套符号系统，和心理学、统计学有相通之处。你可以不信，但值得了解。

5. **误区：需要"天赋"才能学**  
   ✅ 正确：八字是逻辑系统，像学数学一样，谁都能学。需要的是耐心，不是天赋。
