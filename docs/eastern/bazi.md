# 八字基础

> 八字是东方玄学的基础，学会排盘和读干支，其他术数就好学了。

---

## 什么是八字

**八字**（又称四柱命理）是用一个人的出生时间（年/月/日/时）来推算命运的方法。

为什么叫"八字"？
- 年、月、日、时 = **四柱**
- 每一柱有一个天干 + 一个地支 = **两字**
- 四柱 × 两字 = **八字**

例子：
```
年柱    月柱    日柱    时柱
甲子    丙寅    戊辰    庚申
```

## 第一步：学会排盘

**不要手算！** 用排盘工具。

### 推荐排盘工具

- **[文墨天机](https://www.wenmotianji.com/)** —— 专业八字排盘，免费
- **[八字排盘App](https://apps.apple.com/cn/app/八字排盘/id**) —— iOS/Android都有
- **[紫微斗数排盘](https://www.freezd.com/)** —— 同时排八字和紫微

### 排盘需要什么信息？

1. **公历出生日期**（年/月/日）
2. **出生时间**（精确到小时，最好知道分钟）
3. **出生地点**（用来算真太阳时）

> ⚠️ **重要**：时柱很关键，如果出生时间不准，时柱可能排错。

## 第二步：读懂四柱

排出来的盘，你要能看懂每根柱子是什么意思。

### 年柱 = 祖辈/童年/1-16岁

- 代表你的家族背景、祖辈情况
- 也看你小时候的成长环境

### 月柱 = 父母/青年/17-32岁

- 代表你的父母、家庭关系
- 也看你青年时期的运势

### 日柱 = 自己/中年/33-48岁

- **日干 = 你自己**（称为"日主"或"身"）
- **日支 = 你的配偶宫**（婚姻状况）

> 💡 **核心**：日干是你，所有其他干支都是"和你的关系"。

### 时柱 = 子女/晚年/49岁后

- 代表你的子女、晚年运势

## 第三步：认识五行

五行是八字的核心分析工具。

### 五行相生（好的循环）

```
木 → 火 → 土 → 金 → 水 → 木
```

- 木生火（木头可以烧火）
- 火生土（火烧完变成灰土）
- 土生金（土里挖出金属）
- 金生水（金属熔化变成液态）
- 水生木（水浇树，树生长）

### 五行相克（制约/冲突）

```
木 → 土 → 水 → 火 → 金 → 木
```

- 木克土（树根破土）
- 土克水（土可以筑坝挡水）
- 水克火（水可以灭火）
- 火克金（火可以熔化金属）
- 金克木（金属可以砍树）

## 第四步：认识十神（核心中的核心）

**十神**是八字最重要、也最难的概念。它是用"日干"和其他干支的关系来定义的。

### 十神表（以日干为"我"）

假设日干是**甲木**（你是甲木），其他干支和你的关系：

| 五行关系 | 同性（偏） | 异性（正） |
|----------|------------|------------|
| 生我（妈妈） | 偏印（枭神） | 正印 |
| 我生（子女） | 伤官 | 食神 |
| 克我（官杀） | 七杀（偏官） | 正官 |
| 我克（财） | 偏财 | 正财 |
| 同我（兄弟） | 比肩 | 劫财 |

> 💡 **记忆技巧**：
> - "正" = 阴阳不同（异性），"偏" = 阴阳相同（同性）
> - 生我 = 印（妈妈生我），我生 = 食伤（我生孩子）
> - 克我 = 官杀（管我的人），我克 = 财（我追求的东西）

### 十神代表什么？

| 十神 | 代表的事物 |
|------|------------|
| 正印 | 母亲、学业、贵人、稳定 |
| 偏印 | 继母、冷门学问、直觉 |
| 食神 | 才华、口福、温和 |
| 伤官 | 才华、叛逆、锋芒 |
| 正官 | 事业、丈夫、规矩 |
| 七杀 | 压力、野心、危机 |
| 正财 | 稳定收入、妻子 |
| 偏财 | 意外之财、情人 |
| 比肩 | 朋友、竞争 |
| 劫财 | 朋友、破财 |

## 第五步：看日主强弱

**日主强弱** = 你（日干）在八字里是强还是弱。

### 什么影响日主强弱？

1. **得令**：出生月份对你的五行有利？（最重要）
2. **得地**：地支里有你的根（同类五行）？
3. **得势**：天干有其他同类帮你？

### 为什么日主强弱重要？

- **身强**：能担财官（有能力承受好运）
- **身弱**：担不住（好运来了也容易错过，需要印比帮身）

## 常见误区

1. **误区：八字能算出一切**  
   ✅ 正确：八字看趋势，不是定数。努力可以改变很多。

2. **误区：身强就是好，身弱就是不好**  
   ✅ 正确：身强身弱都要看喜用神。身强如果忌神多，也不一定好。

3. **误区：十神意思要背得一字不差**  
   ✅ 正确：理解逻辑，组合时灵活解读（和雷诺曼有点像！）
