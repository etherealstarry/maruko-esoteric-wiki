# 紫微斗数 · 学习笔记

> 东方玄学的另一大体系，和八字并列，但更精细。

---

## 什么是紫微斗数

紫微斗数是中国古代的命理学问，和八字一样用出生时间排盘，但**更精细**——它有14颗主星 + 很多辅星，能看出更细致的性格和运势。

**和八字的区別：**
- 八字：看"趋势"，宏观
- 紫微：看"性格和具体事件"，微观

**核心特点：**
- 用"星"来说事（类似西方占星的行星）
- 12个宫位（类似占星的宫位）
- 可以看"流年"（每年运势）和"流月""流日"

---

## 十四主星

紫微斗数的核心，是这14颗星：

### 紫微系（6颗）

| 星 | 含义 | 关键词 |
|------|------|--------|
| 紫微 | 帝星、领导力 | 尊贵、固执、领导 |
| 天机 | 智慧、变动 | 聪明、善变、策划 |
| 太阳 | 光明、父亲 | 热情、付出、男性 |
| 武曲 | 财星、刚毅 | 财富、固执、行动力 |
| 天同 | 福星、享受 | 温和、懒散、孩子气 |
| 廉贞 | 囚星、感情 | 复杂、感情丰富、艺术 |

### 天府系（8颗）

| 星 | 含义 | 关键词 |
|------|------|--------|
| 天府 | 库星、稳定 | 保守、储蓄、稳定 |
| 太阴 | 财星、母亲 | 阴柔、内敛、女性 |
| 贪狼 | 桃花、欲望 | 多才、桃花、欲望 |
| 巨门 | 暗星、口舌 | 口才、是非、怀疑 |
| 天相 | 印星、辅助 | 温和、辅佐、稳定 |
| 天梁 | 寿星、老人 | 成熟、照顾、清高 |
| 七杀 | 将星、冲劲 | 冲劲、孤独、变动 |
| 破军 | 耗星、改革 | 破旧、创新、不稳定 |

> 💡 **记忆技巧**：先记紫微、天机、太阳、武曲、天同、廉贞（紫微系6颗），再记其他的。

## 十二宫位

紫微盘有12个宫位，和八字、占星的宫位类似：

| 宫位 | 代表领域 |
|------|----------|
| 命宫 | 性格、外貌、一生基调 |
| 兄弟宫 | 兄弟姐妹、朋友 |
| 夫妻宫 | 婚姻、伴侣 |
| 子女宫 | 子女、晚辈 |
| 财帛宫 | 钱财、收入 |
| 疾厄宫 | 健康、疾病 |
| 迁移宫 | 外出、搬家、远方 |
| 仆役宫 | 下属、朋友 |
| 官禄宫 | 事业、职业 |
| 田宅宫 | 房产、家庭 |
| 福德宫 | 内心、福气、精神 |
| 父母宫 | 父母、长辈 |

> 💡 **核心**：命宫最重要，是所有分析的起点。

## 学习步骤

```
1. 先学14主星的含义（最重要）
2. 再学12宫位
3. 学辅星（左辅、右弼、文昌、文曲等）
4. 学四化（化禄、化权、化科、化忌）
5. 学流年流月
```
