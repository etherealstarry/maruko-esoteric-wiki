# 紫微斗数

> 和八字类似，但用的是星曜系统。

---

## 什么是紫微斗数

紫微斗数是中国传统命理学的一种，用**星曜**在十二宫位的分布来推算命运。

**和八字的异同：**
- 相同：都用出生时间排盘，都看命运走势
- 不同：八字用干支+十神，紫微用星曜+宫位

---

## 基础概念

### 主星（最重要）

| 星曜 | 性质 | 代表 |
|------|------|------|
| 紫微 | 帝星 | 领导、尊严 |
| 天机 | 谋臣 | 智慧、策划 |
| 太阳 | 正财 | 父亲、男性贵人 |
| 武曲 | 财星 | 行动、毅力 |
| 天同 | 福星 | 享受、孩子气 |
| 廉贞 | 次桃花 | 感情、艺术 |
| 天府 | 库星 | 稳定、保守 |
| 太阴 | 财星 | 母亲、女性贵人 |
| 贪狼 | 桃花 | 欲望、多才多艺 |
| 巨门 | 暗星 | 口才、是非 |
| 天相 | 印星 | 辅助、体贴 |
| 天梁 | 寿星 | 长辈、法律 |
| 七杀 | 将星 | 冲劲、变动 |
| 破军 | 耗星 | 破旧立新 |
| 文昌 | 文星 | 学业、才华 |
| 文曲 | 文星 | 才艺、桃花 |

### 十二宫位

和占星的宫位类似，代表生活的12个领域：
- 命宫 = 你自己
- 兄弟宫 = 兄弟姐妹
- 夫妻宫 = 婚姻
- 子女宫 = 子女
- 财帛宫 = 财富
- 疾厄宫 = 健康
- 迁移宫 = 外出、远方
- 交友宫 = 朋友、下属
- 事业宫 = 事业
- 田宅宫 = 不动产
- 福德宫 = 精神生活
- 父母宫 = 父母
