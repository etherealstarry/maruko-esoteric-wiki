# 手相 · 学习笔记

> 看手掌的纹路，推断性格、命运、健康。

---

## 什么是手相

手相（Palmistry）是通过观察**手掌纹路、形状、气色**来推断性格和命运的方法。

**核心思想：**
- 手纹不是"天生的"，而是**会变化**的
- 手纹反映你**过去3-5年的状态**
- 不是定数，手纹会随着你的状态改变

---

## 三大主纹

### 1. 生命线（Life Line）

```
从虎口出发，绕大拇指根的弧线
```

| 纹路 | 含义 |
|--------|--------|
| 深长清晰 | 生命力强、健康 |
| 浅短 | 体质弱、易疲劳（不是短命！） |
| 有岛纹（○） | 某段时间健康问题 |
| 断裂 | 人生有重大转折/健康危机 |

> 💡 **误区纠正**：生命线短 ≠ 寿命短。很多长寿的人生命线很短。

### 2. 智慧线（Head Line）

```
从虎口出发，横向穿过手掌
```

| 纹路 | 含义 |
|--------|--------|
| 深长清晰 | 思维清晰、理智 |
| 和生命线起点分开 | 独立、叛逆、早离家 |
| 和生命线连在一起 | 谨慎、依赖、晚独立 |
| 下垂到月丘 | 想象力丰富、感性 |
| 上扬到火星丘 | 行动力、现实 |

### 3. 感情线（Heart Line）

```
从小指下方出发，横向到食指/中指下方
```

| 纹路 | 含义 |
|--------|--------|
| 深长清晰 | 重感情、专一 |
| 断续 | 感情波折、易受伤 |
| 末端分叉 | 犹豫不决、感情复杂 |
| 有岛纹 | 某段感情有秘密/心碎 |

## 其他重要纹路

### 命运线（ Fate Line）

```
从手掌底部向上，穿过智慧线/感情线
```

- 有命运线 = 人生有"主线任务"，事业心强
- 没有命运线 = 随遇而安，不强求
- 命运线断断续续 = 职业/人生方向多次变化

### 婚姻线（Marriage Line）

```
小指下方，短横线（可以有1-3条）
```

- 一条深长 = 一段稳定婚姻/感情
- 多条 = 感情经历丰富
- 末端分叉 = 分居、离异倾向

### 太阳线（Sun Line）

```
从无名指下方，向下的短线
```

- 有太阳线 = 有名声、才华被认可
- 没有 = 默默无闻也可以（不是"不好"）

## 手掌形状（掌型）

| 掌型 | 元素 | 性格 |
|--------|--------|--------|
| 土型（方掌、指短） | 土 | 务实、稳定、慢热 |
| 水型（长掌、指长） | 水 | 感性、直觉强、艺术 |
| 火型（掌厚、指尖尖） | 火 | 热情、冲动、领导力 |
| 风型（掌薄、指长） | 风 | 理性、沟通强、思维 |

> 💡 **记忆技巧**：掌型 = 西方四元素，和塔罗/占星对应！

## 学习步骤

```
1. 先学三大主纹（生命线、智慧线、感情线）
2. 再学其他纹路（命运线、婚姻线等）
3. 学掌型（和西方四元素对应！）
4. 实战：看自己的手，对照；看朋友，验证
```

## 常见误区

1. **误区：生命线短 = 短命**  
   ✅ 正确：生命线看"生命力"，不是"寿命"。短命看"耳垂""法令纹"等，而且命运可以改变。

2. **误区：手纹是天生的，不会变**  
   ✅ 正确：手纹会随状态改变。比如大病后，生命线会出现岛纹；状态变好后，岛纹会消失。

3. **误区：左手先天，右手后天**  
   ✅ 正确：这个说法不准确。正确的是——**男生看左手，女生看右手**（或者以惯用手为主）。
