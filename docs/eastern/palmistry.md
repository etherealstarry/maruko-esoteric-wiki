# 手相 · 完全学习笔记

> 手纹会变——它反映的是你过去3-5年，不是一生。

---

## 什么是手相

手相（Palmistry）通过观察手掌纹路、形状、气色推断性格和命运。

**最重要的事实：手纹会变！**
- 大病后生命线出现岛纹
- 状态好转后岛纹消失
- 手纹是你选择的结果，不是命运

---

## 三大主纹

### 1. 生命线

从虎口出发，绕大拇指根。

| 纹路 | 含义 |
|------|------|
| 深长清晰 | 生命力强 |
| 浅短 | 体质弱（不是短命！） |
| 有岛纹 | 某段健康出问题 |
| 断裂 | 人生重大转折 |

> 💡 **纠正**：生命线短 ≠ 短命。很多长寿人手生命线很短。

### 2. 智慧线

从虎口出发，横穿手掌。

| 纹路 | 含义 |
|------|------|
| 深长 | 思维清晰 |
| 与生命线分开 | 独立、早离家 |
| 与生命线连 | 谨慎、依赖 |
| 下垂 | 想象力丰富 |

### 3. 感情线

从小指下方横到食指/中指下方。

| 纹路 | 含义 |
|------|------|
| 深长清晰 | 重感情、专一 |
| 断续 | 感情波折 |
| 末端分叉 | 感情复杂 |

## 掌型（四元素对应）

| 掌型 | 元素 | 性格 |
|------|------|------|
| 方掌指短 | 土 | 务实、稳定 |
| 长掌指长 | 水 | 感性、艺术 |
| 掌厚指尖 | 火 | 热情、冲动 |
| 掌薄指长 | 风 | 理性、思维 |

> 💡 手掌的西方四元素对应！和塔罗/占星相通。

---

## 学习方法

### 入门（1天）
1. 看自己的三大主纹
2. 判断自己的掌型
3. 男左女右（或用惯用手）

### 进阶（1周）
1. 看朋友的
2. 记录+验证
3. 手纹变化（3个月后对比）

---

## 入门→精通路线

```
入门（1天）→ 看三大主纹
进阶（1周）→ 看朋友，验证
精通（1月）→ 学命运线/婚姻线/太阳线，跟踪自己的手纹变化
```
