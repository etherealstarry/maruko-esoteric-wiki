# 山 · 修行与内炼

> "山"是东方玄学里最神秘的部分——修行、气功、内丹，超越普通人的境界。

---

## 什么是"山"

"山"是道家五术（山医命相卜）之首，代表**修行、超越、成仙**的路径。

名字来源：修行者往往住在山里，远离人群。

**包含的内容：**
- 内丹术（炼精化气、炼气化神、炼神还虚）
- 气功（气机运行、周天）
- 导引术（八段锦、易筋经）
- 符咒（祝由科、画符）
- 禅定（打坐、入静）

---

## 内丹术基础

内丹是道家修行的核心方法，目标是**"成仙"（超越生死）**。

### 内丹三步

```
炼精化气 → 炼气化神 → 炼神还虚
```

| 阶段 | 含义 | 现代理解 |
|------|------|----------|
| 炼精化气 | 把"精"（生命能量）炼化成"气" | 养生、健康、活力 |
| 炼气化神 | 把"气"炼化成"神"（意识） | 精神清晰、直觉开发 |
| 炼神还虚 | 把"神"回归"虚"（空性） | 超越自我、灵性觉醒 |

> 💡 **现代人怎么理解**：内丹不是"迷信"，而是一套**身心修炼系统**，和瑜伽、冥想有相通之处。

## 气功基础

气功是用**呼吸、动作、意念**来调节"气"（生命能量）的方法。

### 气功的核心

```
调身（姿势） + 调息（呼吸） + 调心（意念） = 气功
```

### 常见气功

| 气功 | 特点 | 适合谁 |
|------|------|--------|
| **八段锦** | 8个动作，简单 | 初学者、日常养生 |
| **易筋经** | 拉伸为主，强筋骨 | 需要强身的人 |
| **五禽戏** | 模仿5种动物 | 有趣、适合长期练 |
| **站桩** | 静止站立，最难但最有效 | 进阶者 |

> 💡 **推荐入门**：八段锦。B站有视频，每天15分钟，坚持一个月有感觉。

## 符咒（祝由科）

符咒是"山"里最神秘、也最有争议的部分。

### 符（Fulu）

- 用朱砂在黄纸上画"符"
- 符是一种**能量编码**，不是随便画的
- 需要"灌顶"或"传承"才能有效（说法）

### 咒（Zhou）

- 念诵特定的音（真言、咒语）
- 核心：**音的频率**会影响身体和气场
- 现代理解：咒语是一种"声波疗愈"

### 祝由科

- 古代"心理疗愈"+"能量疗愈"的结合
- 用符、咒、意念来治病
- 《黄帝内经》里就有记载，不是"迷信"

> ⚠️ **注意**：符咒部分争议很大，建议你先学"山"的其他部分（气功、八段锦），符咒可以后学，或者只做文化了解。

## 禅定与打坐

### 打坐的正确方法

1. **姿势**：盘腿（单盘/双盘/散盘都行），背直，下颌微收
2. **呼吸**：自然呼吸 → 深呼吸 → 腹式呼吸
3. **意念**：专注在"下丹田"（肚脐下3寸）
4. **时间**：初学者5-10分钟，慢慢加到30分钟

### 常见境界（内景）

| 境界 | 体验 |
|--------|------|
| 入静 | 杂念减少，身体放松 |
| 气感 | 感觉到"气"在身体里动 |
| 光感 | 闭眼看到光（内光） |
| 性光 | 深层入定，超越自我感 |

> 💡 **重要**：打坐不要"追求境界"，越追求越得不到。放松，自然来。

## 学习步骤

```
第1-2周：动功（八段锦）
  → 每天15分钟
  → B站跟着视频学
  → 感受身体的变化

第3-4周：站桩
  → 每天站10-15分钟
  → 感受"气感"（手脚发热、发麻）

第5-6周：打坐
  → 每天10-20分钟
  → 学腹式呼吸
  → 专注下丹田

第7-8周：进阶
  → 学"小周天"（气在任督二脉运行）
  → 读《道德经》
  → 开始"山"的深层理解

长期：
  → 八段锦/站桩/打坐，成为日常
  → 读《庄子》
  → 符咒部分（可选，文化了解）
```

## 常见误区

1. **误区：气功是迷信**  
   ✅ 正确：气功是**身心调节技术**，和瑜伽、冥想同类。中国官方也认可（有"健身气功协会"）。

2. **误区：内丹是"修仙"**  
   ✅ 正确：内丹的现代理解是**身心健康的高级状态**，不是真的"成仙"。

3. **误区：打坐越久越好**  
   ✅ 正确：初学者5-10分钟就够了，慢慢加。强行久坐会"走火入魔"（其实是身体受伤）。

4. **误区：符咒可以"驱鬼"**  
   ✅ 正确：符咒的文化价值大于实用价值。建议先学其他部分，符咒当文化了解。
