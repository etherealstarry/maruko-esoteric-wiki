# 东方玄学

> "天人合一" —— 华夏文明的底层逻辑

---

## 什么是东方玄学

东方玄学是一个广义词，包括：
- **命理学**：八字、紫微斗数（看先天命运）
- **占卜术**：易经（六爻、梅花易数）、奇门遁甲（预测未来）
- **相术**：面相、手相、风水
- **修行体系**：内丹、气功、冥想

这个网站主要关注**命理学和占卜术**——因为它们最实用，也最适合系统学习。

---

## 核心概念：天人合一

东方玄学的底层逻辑是：

> **"天人合一"** —— 人和宇宙是自然对应的

具体表现：
- **天干地支** = 把时间编码成符号（年/月/日/时都有对应的干支）
- **五行生克** = 事物之间的相互作用关系
- **阴阳** = 任何事物都有两面

占卜/排盘，就是在读"天象"和"人事"之间的对应关系。

---

## 学习顺序（推荐）

```
1. 八字基础 → 先学天干地支，这是所有东方玄学的基础
2. 八字进阶 → 十神、大运、流年
3. 紫微斗数 → 类似八字，但用的是星曜系统
4. 易经占卜 → 六爻或梅花易数，适合问具体事件
5. 奇门遁甲 → 高阶，用于决策和择时
```

## 基础中的基础：天干地支

不管学哪个东方玄学板块，**天干地支一定要先背熟**。

### 十天干

```
甲(jiǎ) 乙(yǐ) 丙(bǐng) 丁(dīng) 戊(wù)
己(jǐ) 庚(gēng) 辛(xīn) 壬(rén) 癸(guǐ)
```

对应五行：
- 木：甲、乙
- 火：丙、丁
- 土：戊、己
- 金：庚、辛
- 水：壬、癸

### 十二地支

```
子(zǐ) 丑(chǒu) 寅(yín) 卯(mǎo) 辰(chén) 巳(sì)
午(wǔ) 未(wèi) 申(shēn) 酉(yǒu) 戌(xū) 亥(hài)
```

对应生肖：
- 子=鼠，丑=牛，寅=虎，卯=兔
- 辰=龙，巳=蛇，午=马，未=羊
- 申=猴，酉=鸡，戌=狗，亥=猪

> 💡 **学习技巧**：地支对应生肖好记，先记生肖再记地支名。

## 东西方共通点

我在学的时候发现，东西方玄学有很多惊人的相似：

| 西方 | 东方 | 共通点 |
|------|------|--------|
| 四元素（土水火风） | 五行（木火土金水） | 元素系统 |
| 占星（行星→星座→宫位） | 八字（干支→十神→大运） | 时间编码成符号 |
| "As above, so below" | "天人合一" | 天人对应 |
| 塔罗牌象征 | 卦象象征 | 象征语言 |
| 阴阳（很多牌隐含） | 阴阳（底层逻辑） | 二元结构 |
