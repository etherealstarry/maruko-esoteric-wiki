# 读书笔记

> 每本书都按固定模板记笔记，方便复习和查阅。

---

## 为什么要做读书笔记

玄学书籍信息量大，不做笔记很容易：
- 看了就忘
- 找不到之前看到的重点
- 无法追踪自己的学习进度

这个板块的作用是：**把书读薄，把知识变成自己的**。
