# 学习路线

> 从零开始，怎么系统学玄学？这份路线是根据我的学习进度和目标制定的。

---

## 为什么要有学习路线

玄学体系庞大，如果没有顺序地学，很容易：
- 同时学太多，哪个都学不深
- 基础不牢，进阶时卡住
- 不知道自己学到哪了

这份路线的作用是：**告诉你先学什么、为什么、学到什么程度可以进阶**。

---

## 整体学习顺序

```
阶段一：入门（1-2个月）
├── 西方：雷诺曼（简单，适合入门）
└── 东方：八字基础（天干地支、排盘）

阶段二：建立体系（2-3个月）
├── 西方：塔罗牌（伟特系）
├── 西方：占星基础（行星、星座、宫位）
└── 东方：八字进阶（十神、大运）

阶段三：深化+实战（长期）
├── 西方：占星进阶（合盘、流年）
├── 东方：紫微斗数
└── 共通点研究：找东西方相通的逻辑

阶段四：高阶
├── 奇门遁甲
├── 易经占卜（六爻、梅花易数）
└── 建立自己的解读体系
```

## 阶段一：入门（当前阶段）

### 西方：雷诺曼

**为什么先学雷诺曼？**
- 牌意简单直接，不像塔罗那么抽象
- 适合占卜具体事件（"这件事会怎样"）
- 入门快，容易有成就感

**学习步骤：**
1. 认识牌意（先学核心35张）
2. 看牌与牌的组合
3. 实战练习（每天1-2个案例）

**资源：**
- 《The Complete Lenormand Oracle Handbook》—— 推荐入门书
- [雷诺曼学习笔记](/western/lenormand) —— 我的笔记

---

### 东方：八字基础

**为什么学八字？**
- 东方玄学的基础，很多其他术数都用到干支
- 可以看长期趋势，不只问一件事
- 有系统性，逻辑清晰

**学习步骤：**
1. 天干地支（背熟！）
2. 排盘（学会用工具排，理解每个部分是什么）
3. 十神（这是核心概念）
4. 大运流年（时间维度）

**资源：**
- [八字学习笔记](/eastern/bazi) —— 我的笔记（建设中）

## 学习原则

1. **先实战，再理论** —— 学了一点就试试，不要等"学完"再动手
2. **记录复盘** —— 每次占卜都记下来，回头看准不准
3. **东西方交叉学** —— 不要一口气只学西方或只学东方，换着学不容易腻
4. **接受"不准"** —— 初期不准是正常的，复盘才有进步
