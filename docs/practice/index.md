# 实战记录

> 占卜不是"算命"，是练习直觉和解读能力。
> 记录每一次实战，回头复盘，才能进步。

---

## 为什么要记录实战

- **追踪准确率**：知道自己哪类问题准，哪类不准
- **发现模式**：同样的问题，牌面/卦象有没有重复的模式？
- **积累案例**：以后遇到类似问题，可以回头参考
- **复盘成长**：一个月后看自己的解读，会发现进步

## 复盘方法

1. **等结果出来再复盘**（不要当天就复盘，会主观）
2. **诚实记录**（不准就是不准，不要美化）
3. **找原因**（不准是因为牌意理解错了？还是问题问得不好？）
4. **调整方法**（下次遇到类似问题，怎么改进？）
