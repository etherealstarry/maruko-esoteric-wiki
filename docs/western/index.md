## 什么是西方神秘学

西方神秘学（Western Esotericism）是一个庞大的知识体系，包括：
- **占卜系统**：塔罗、雷诺曼、占星
- **魔法传统**：赫密斯主义、卡巴拉、仪式魔法
- **灵性修行**：黄金黎明、神智学、新时代运动

这个网站主要关注**占卜系统**——因为它们最实用，也最适合入门。

---

## 核心概念：天人对应

西方神秘学的底层逻辑是赫密斯主义的核心理念：

> **"As above, so below"** —— 如其在上，如其在下

意思是：**宏观（天/星/宇宙）和微观（人/事/物）是同构的**。

占星看星盘，塔罗看牌阵，都是在读这个"映射关系"。

---

## 学习顺序（推荐）

```
1. 雷诺曼 → 最简单，适合入门
2. 塔罗牌 → 进阶，更抽象，更深度
3. 占星术 → 体系最大，需要长期学习
4. 西方数术 → 数字神秘学，可以作为补充
```

## 东西方共通点

我发现西方神秘学和东方玄学有很多相通的地方：
- 都用**象征语言**（不直接说，用符号让你"悟"）
- 都有**时空对应系统**（占星用星盘，八字用干支）
- 都相信**天人对应**（人是小宇宙，宇宙是大的人）
