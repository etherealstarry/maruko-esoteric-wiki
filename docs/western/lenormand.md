# 雷诺曼占卜

> 简单、直接、准。适合占卜具体事件。

---

## 什么是雷诺曼

雷诺曼（Lenormand）是一种卡牌占卜系统，起源于19世纪的法国，以女占卜师**玛丽·安妮·雷诺曼**的名字命名。

和塔罗不同，雷诺曼：
- **牌意简单直接**（不像塔罗那么抽象）
- **看牌与牌的组合**（两张或三张牌一起解读）
- **适合占卜具体事件**（"这件事会怎样？"）

---

## 解读方法

雷诺曼的核心不是单张牌的意思，而是**牌与牌的组合**。

### 基础：两张牌读法

两张牌一起读，通常是**"牌A + 牌B" = 牌A修饰牌B**

例子：
- **Rider（骑士）+ House（房子）** = 消息到家 = 收到家信/家人来访
- **Clover（四叶草）+ Tree（树）** = 幸运的健康 = 身体好转/康复
- **Snake（蛇）+ Bear（熊）** = 麻烦的权力 = 有权势的人找麻烦

### 进阶：三张牌读法

三张牌 = **过去 + 现在 + 未来** 或 **主题 + 修饰1 + 修饰2**

例子（问：这段关系会怎样？）：
- **Bouquet（花束）- Child（孩子）- Dog（狗）**
- 解读：美好的新开始 + 有朋友的支持
- 结论：新的关系，有朋友祝福

## 常用牌阵

### 1. 三张牌（最简单）

```
[过去] [现在] [未来]
```

适合：快速问一件事的结果

### 2. 五张牌（十字牌阵）

```
      [2]
  [1] [3] [4]
      [5]
```

- 1 = 过去
- 2 = 阻碍
- 3 = 现在/核心
- 4 = 未来/建议
- 5 = 结果

### 3. 九张牌（方形牌阵）

适合问复杂问题，可以看到更多细节。

## 常见误区

1. **误区：牌意要背得一字不差**  
   ✅ 正确：理解核心意思，组合时灵活解读

2. **误区：一定要用传统牌（小木人牌）**  
   ✅ 正确：任何雷诺曼牌组都可以，选你喜欢的画面

3. **误区：牌阵越复杂越好**  
   ✅ 正确：三张牌往往就够了，复杂牌阵容易信息过载
