# 占星术

> 看星盘，读懂性格和长期趋势。

---

## 什么是占星术

占星术（Astrology）是通过观察天体运行来预测人间事务的体系。

**西方占星术**主要看：
- **本命盘**（Natal Chart）—— 性格、天赋、人生主题
- **流年**（Transits）—— 当前正在发生什么
- **合盘**（Synastry）—— 两个人之间的关系

---

## 基础概念

### 行星（Planets）

| 行星 | 代表 |
|------|------|
| 太阳 | 自我、意志、父亲 |
| 月亮 | 情绪、内在需求、母亲 |
| 水星 | 思维、沟通 |
| 金星 | 爱情、审美、价值 |
| 火星 | 行动、欲望、冲突 |
| 木星 | 扩张、幸运、哲学 |
| 土星 | 限制、责任、时间 |
| 天王星 | 突变、革新 |
| 海王星 | 梦幻、灵性 |
| 冥王星 | 转化、深层力量 |

### 星座（Signs）

12星座，每个星座有对应的元素和模式：

| 星座 | 元素 | 模式 |
|------|------|------|
| 白羊 | 火 | 开创 |
| 金牛 | 土 | 固定 |
| 双子 | 风 | 变动 |
| 巨蟹 | 水 | 开创 |
| 狮子 | 火 | 固定 |
| 处女 | 土 | 变动 |
| 天秤 | 风 | 开创 |
| 天蝎 | 水 | 固定 |
| 射手 | 火 | 变动 |
| 摩羯 | 土 | 开创 |
| 水瓶 | 风 | 固定 |
| 双鱼 | 水 | 变动 |

### 宫位（Houses）

12宫位，代表生活的12个领域：
- 1宫 = 自我、外表
- 7宫 = 关系、婚姻
- 10宫 = 事业、社会形象
- 等等

---

## 学习步骤

```
1. 学行星、星座、宫位的基础含义
2. 学本命盘解读（太阳、月亮、上升最重要）
3. 学相位（行星之间的关系）
4. 学流年（当前星象对个人盘的影响）
5. 学合盘（关系占星）
```
