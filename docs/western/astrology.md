# 占星术 · 完全学习笔记

> 看星盘，读懂性格和长期趋势。西方神秘学的核心体系之一。

---

## 什么是占星术

占星术（Astrology）是通过观察天体运行来理解和预测人间事务的体系。

**注意：** 占星不是"星座运势"（那种每天/每周的运势是娱乐化的），而是**基于你出生时刻的精确星盘**来做分析。

**西方占星术主要看：**
- **本命盘**（Natal Chart）—— 性格、天赋、人生主题
- **流年**（Transits）—— 当前正在发生什么
- **合盘**（Synastry）—— 两个人之间的关系

---

## 核心三要素

占星术的"语言"由三样东西组成：**行星 + 星座 + 宫位**

可以理解成一个句子：
> **行星**（主语：谁？）+ **星座**（副词：怎么行动？）+ **宫位**（宾语：在哪里？）

例子：
- **太阳在狮子座，在第10宫** = 你的核心自我（太阳）以狮子座的方式（自信、表现欲）在事业领域（第10宫）表达

## 宫位（Houses）—— 发生在哪里？

宫位是星盘上12个扇形区域，代表生活的12个领域。

| 宫位 | 领域 | 关键词 |
|------|------|--------|
| 第1宫 | 自我、外表 | 第一印象、外貌、自我认同 |
| 第2宫 | 金钱、价值 | 收入、价值观、安全感 |
| 第3宫 | 沟通、学习 | 兄弟姐妹、短途旅行、学习 |
| 第4宫 | 家庭、根源 | 父母、家庭、晚年 |
| 第5宫 | 爱情、创造 | 恋爱、孩子、娱乐、创作 |
| 第6宫 | 工作、健康 | 日常工作、健康、服务 |
| 第7宫 | 关系、婚姻 | 伴侣、合作、公开敌人 |
| 第8宫 | 转化、共享资源 | 性、死亡、别人的钱、深层心理 |
| 第9宫 | 哲学、远方 | 高等教育、出国、信仰、法律 |
| 第10宫 | 事业、社会形象 | 职业、名誉、母亲（现代） |
| 第11宫 | 朋友、希望 | 朋友、团体、愿望、未来 |
| 第12宫 | 潜意识、隐秘 | 潜意识、隔离、灵性、隐秘敌人 |

> 💡 **记忆技巧**：1-6宫是"个人领域"（自我→日常）；7-12宫是"外部领域"（关系→集体→潜意识）。

## 相位（Aspects）—— 行星之间怎么互动？

相位是行星之间的角度关系，代表能量是和谐还是冲突。

| 相位 | 角度 | 性质 | 含义 |
|------|------|------|------|
| 合相（Conjunction） | 0° | 混合 | 能量融合，强烈 |
| 六分相（Sextile） | 60° | 和谐 | 机会，需要行动 |
| 四分相（Square） | 90° | 冲突 | 张力，推动成长 |
| 三分相（Trine） | 120° | 和谐 | 天赋，容易，但可能懒惰 |
| 对分相（Opposition） | 180° | 冲突 | 关系，需要平衡 |

> 💡 **记忆技巧**：和谐相位（三分、六分）= 能量顺畅；冲突相位（四分、对分）= 能量有张力，但也推动成长。没有"好"或"坏"，只是不同。

## 本命盘解读步骤

```
1. 看太阳、月亮、上升（"三巨头"）
   → 太阳 = 核心自我
   → 月亮 = 情绪需求
   → 上升 = 外在表现、第一印象

2. 看行星落在哪些星座（"行星在星座"）
   → 行星的能量以什么方式表达？

3. 看行星落在哪些宫位（"行星在宫位"）
   → 行星的能量在哪个生活领域发挥作用？

4. 看相位（行星之间的关系）
   → 哪些能量和谐？哪些有冲突？

5. 综合解读
   → 把以上信息整合成一个完整的"人物画像"
```

## 流年（Transits）—— 当前正在发生什么？

流年是用**当前星象**和你的**本命盘**做比较，看现在正在经历什么。

**重要流年（值得关注的）：**

| 流年 | 周期 | 影响 |
|------|------|------|
| 土星回归 | 约29岁 | 成熟、责任、人生转折点 |
| 天王星四分 | 约21岁、约63岁 | 突变、叛逆、打破常规 |
| 冥王星四分 | 约37-43岁 | 深层转化、危机与重生 |
| 木星过本命太阳 | 约12年一次 | 扩张、机会、成长 |

## 学习步骤

```
第1-2周：基础概念
  → 行星、星座、宫位的基础含义
  → 每天学3-4个，做笔记

第3-4周：本命盘解读
  → 先学"三巨头"（太阳、月亮、上升）
  → 再学其他行星在星座/宫位
  → 学相位

第5-6周：实战
  → 给自己排盘，解读自己的星盘
  → 给朋友排盘，验证准确率
  → 记录案例

第7-8周：流年
  → 学流年基础
  → 看自己当前的流年
  → 预测未来趋势

长期：
  → 读进阶书籍
  → 学合盘（关系占星）
  → 持续实战
```

## 常见误区

1. **误区：占星就是"你是什么星座"**  
   ✅ 正确：太阳星座只是星盘的1/100。完整的本命盘有10颗行星、12个星座、12个宫位、无数相位。

2. **误区：占星能预测一切**  
   ✅ 正确：占星显示"能量趋势"，不是定数。自由意志可以改变很多。

3. **误区：土星都是不好的**  
   ✅ 正确：土星代表限制，但也代表成熟和责任。土星过运往往是"成长最快"的时期。

4. **误区：需要会算命才能学占星**  
   ✅ 正确：占星是符号系统，像学语言一样，谁都能学。
