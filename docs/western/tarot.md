# 塔罗牌

> 深层心理、灵性成长、复杂问题的占卜工具。

---

## 什么是塔罗牌

塔罗牌（Tarot）是一套78张牌的占卜系统，起源于15世纪的欧洲。

**和雷诺曼的区别：**
- 雷诺曼：简单直接，适合具体事件
- 塔罗：更抽象，适合深层心理、灵性成长、复杂问题

---

## 牌组介绍

塔罗牌分两部分：

### 大阿卡纳（Major Arcana）22张

代表人生的重要课题、精神成长的关键节点。

核心牌：
- 0 愚者（The Fool）—— 新的开始、冒险
- I 魔术师（The Magician）—— 创造力、行动力
- III 女皇（The Empress）—— 丰收、母性
- VII 战车（The Chariot）—— 意志、胜利
- XII 倒吊人（The Hanged Man）—— 牺牲、换个角度
- XXI 世界（The World）—— 完成、圆满

### 小阿卡纳（Minor Arcana）56张

分四花色（类似扑克牌）：
- **权杖（Wands）** = 火 = 行动、创意
- **圣杯（Cups）** = 水 = 情感、关系
- **宝剑（Swords）** = 风 = 思维、冲突
- **星币（Pentacles）** = 土 = 物质、现实

---

## 学习步骤

```
1. 先学大阿卡纳22张（最重要）
2. 再学小阿卡纳（可以按数字含义+花色含义组合理解）
3. 学牌阵（三张牌、凯尔特十字等）
4. 实战练习
```

## 推荐牌组

**伟特塔罗（Rider-Waite）** —— 最适合入门
- 画面有象征意义，容易理解
- 几乎所有教程都用这套
- 买第一套牌的首选
