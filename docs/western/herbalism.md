# 草药学 · 学习笔记

> 植物和灵性的连接，西方神秘学的重要分支。

## 什么是草药学

草药学（Herbalism / Hedge Witchery）是用植物的**能量**来疗愈、保护和占卜的学问。

**和中医草药学的区别：**
- 中医：看四气五味、归经，偏重"药性"
- 西方：看植物灵（Plant Spirit）、元素对应、月相，偏重"灵性"

**核心想法：** 植物不是"死的药材"，而是**有意识的存在**。你可以和植物"对话"。

## 实用方法

### 1. 草药茶（Tea Magic）

最简单的入门方法。

**步骤：**
1. 选对应你目标的草药
2. 烧水，等水开
3. 把草药放进去，盖上盖子（"锁住能量"）
4. 默念你的意图
5. 喝的时候，感受能量

**配方示例：**
- **爱情茶**：玫瑰 + 肉桂 + 蜂蜜
- **保护茶**：大蒜 + 生姜 + 柠檬
- **直觉茶**：茉莉 + 薄荷 + 月见草

### 2. 草药包（Herb Bundle）

把干燥的草药包在布里，放在枕头下/口袋里/门口。

**用途：**
- 枕头下 = 助眠（薰衣草）、预知梦（月见草）
- 口袋里 = 幸运（四叶草）、保护（大蒜）
- 门口 = 保护家宅（迷迭香、鼠尾草）

### 3. 烟熏（Smudging）

用干燥的草药（通常是鼠尾草、雪松）燃烧，烟熏房间/人。

**用途：**
- 净化空间（去掉负能量）
- 净化自己（仪式前/后）
- 送走不好的东西

> ⚠️ **注意**：烟熏是北美原住民的仪式，用的时候要尊重起源，不要"文化挪用"。

### 4. 植物占卜（Botanical Divination）

用植物的形态、生长情况来占卜。

**方法：**
- **茶叶占卜（Tasseography）**：喝完茶，看茶叶在杯底的图案
- **树枝占卜（Radionics）**：用树枝的弯曲来判断"是/否"
- **植物生长**：种的植物长得好 = 能量对；长不好 = 需要调整

## 常见误区

1. **误区：草药学就是"喝花草茶"**  
   ✅ 正确：花草茶只是入门。草药学包含能量工作、占卜、仪式。

2. **误区：所有植物都能随便用**  
   ✅ 正确：有些植物有毒（乌头、颠茄），有些会过敏。用之前先查。

3. **误区：需要"天赋"才能和植物对话**  
   ✅ 正确：植物一直在"说话"，只是你没注意。去阳台坐5分钟，什么都不做，感受植物，就有了。
