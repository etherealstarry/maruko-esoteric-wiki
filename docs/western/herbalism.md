## 什么是草药学

草药学（Herbalism / Green Witchery）是用植物的**能量**来疗愈、保护和占卜的学问。

**和中医草药学的核心区别：**

| | 西医草药 | 中医草药 | 西方草药学 |
|---|---------|---------|-----------|
| 核心 | 有效成分 | 四气五味归经 | 植物灵/能量 |
| 用法 | 提取物 | 煎煮配伍 | 泡茶/烟熏/仪式 |

**核心想法：** 植物不是"死的药材"，而是**有意识的存在**。你可以和植物"对话"——不是真的说话，而是感受它。

## 实用方法

### 方法1：草药茶（最安全，最日常）

**步骤：**
1. 选对应目标的草药
2. 烧水，等水开
3. 入草药，盖盖子——"锁住能量"
4. 静置3-5分钟，默念意图
5. 喝的时候，专注感受

**配方速查：**

| 目标 | 配方 | 行星 |
|------|------|------|
| 提升自信 | 迷迭香 + 肉桂 + 柠檬皮 | 太阳 |
| 情绪平衡 | 茉莉 + 洋甘菊 + 蜂蜜 | 月亮 |
| 考前清醒 | 薄荷 + 薰衣草 + 柠檬 | 水星 |
| 自爱/爱情 | 玫瑰 + 薰衣草 + 蜂蜜 | 金星 |
| 保护/驱散 | 生姜 + 肉桂 + 柠檬 | 火星 |
| 好运/扩张 | 肉桂 + 丁香 + 橙皮 | 木星 |
| grounding | 鼠尾草 + 雪松（闻，不喝） | 土星 |

### 方法2：草药包

干草药包在布里，放在枕头下/口袋里/门口。

**制作步骤：**
1. 选布（天然材质最好：棉/麻）
2. 放干草药，加一撮盐（净化）
3. 绑好，放在目标位置
4. 满月夜拿出来晒月光（充能）

| 位置 | 草药 | 用途 |
|------|------|------|
| 枕头下 | 薰衣草 + 洋甘菊 | 助眠 |
| 枕头下 | 茉莉 + 月见草 | 预知梦 |
| 口袋 | 迷迭香 + 肉桂 | 考试/面试好运 |
| 门口 | 迷迭香 + 鼠尾草 + 盐 | 保护家宅 |

### 方法3：烟熏（Smudging）

用干燥草药燃烧，烟熏房间/人/物品。

**常用烟熏草药：**

| 草药 | 用途 | 气味 |
|------|------|------|
| 白鼠尾草 | 强力净化 | 浓郁、土味 |
| 雪松 | 保护、祝福 | 木质、温暖 |
| 迷迭香 | 净化、记忆 | 清新、草本 |
| 薰衣草 | 平和、放松 | 花香 |
| 甜草 | 召唤好能量 | 甜味 |

> ⚠️ 烟熏是北美原住民的传统仪式，用的时候要尊重起源，了解它的文化背景。

### 方法4：植物占卜

**茶叶占卜（Tasseography）：** 喝完整杯散茶，看杯底茶叶的形状。

| 茶叶形状 | 含义 |

| 直线 | 道路、方向、前进 |
| 心形 | 爱情、情感 |
| 鸟形 | 消息、自由 |
| 蛇形 | 麻烦、警惕 |

## 学习经验

### 入门阶段（第1-2周）

**目标：** 认识10种基础草药，学会泡第一杯茶。

**方法：**
1. 买5种干草药（薰衣草、玫瑰、薄荷、迷迭香、洋甘菊）
2. 每天泡一种，喝之前闻、看、摸——用感官认识它
3. 记录感受（"薰衣草让我觉得平静，有点困"）

**常见坑：**
- 不要一次买太多（先用5种，熟悉了再加）
- 不是所有植物都能泡茶喝，有些有毒，买之前查一下
- 从花草茶开始，别从烟熏开始（烟熏需要技巧）

### 进阶阶段（第3-4周）

**目标：** 记住植物和行星/元素的对应，学会2-3种实用方法。

**方法：**
1. 把10种草药的"行星+元素"表格贴墙上，每天看
2. 学做草药包（从简单的开始）
3. 开始"草药日记"——记录每一株植物的感受

### 精通阶段（第5-8周）

**目标：** 开发自己的"植物直觉"——不是查表，而是感受。

**方法：**
1. 去花店/花园，不看标签，先去"感受"植物
2. 然后查标签，看你的感受对不对
3. 做自己的草药配方（不查书，自己组合，实验）

## 记忆方法

### 方法1：行星性格法

不硬背表格，用"行星性格"来记：

```
太阳植物 = 自信的、明亮的、领导型的（迷迭香/金盏花）
月亮植物 = 柔和的、情绪的、阴性的（茉莉/百合）
水星植物 = 清晰的、沟通的、思维的（薄荷/薰衣草）
金星植物 = 美的、甜的、爱情的（玫瑰/草莓）
火星植物 = 辣的、有刺的、保护的（辣椒/大蒜）
木星植物 = 扩大的、木本的、慷慨的（肉桂/橡树）
土星植物 = 稳重的、深沉的、限制的（雪松/松树）
```

### 方法2：五感记忆法

不是背表格，而是**用感官记住植物**：

1. **看**：颜色、形状
2. **摸**：质感（薰衣草干干的，薄荷凉凉的）
3. **闻**：气味（闭上眼睛闻）
4. **尝**：味道（薰衣草微苦，玫瑰甜）
5. **感受**：它的能量给你的感觉

### 方法3：草药日记法

准备一个本子，每学一种植物，写：

```
植物：薰衣草
行星：水星
元素：风
我感受到的：平静、放松、有点想睡觉
我第一次用它：泡茶，喝了10分钟后真的觉得很放松
```

## 常见误区

1. **误区：草药学就是"喝花草茶"**  
   ✅ 正确：花草茶只是入门。草药学包括能量工作、占卜、仪式、和植物灵对话。

2. **误区：所有植物都能泡茶**  
   ✅ 正确：有些有毒（乌头、颠茄），有些会过敏。用之前先查。只喝明确标注"可食用"的。

3. **误区：需要天赋才能和植物交流**  
   ✅ 正确：去阳台坐5分钟，什么都不做，感受植物——这就是"交流"。
