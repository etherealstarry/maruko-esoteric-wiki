## 什么是西方数术

西方数术（Numerology）是用**出生日期**和**名字**来推算性格和命运趋势。起源于古希腊毕达哥拉斯学派。

**核心想法：** 数字不是随机的，每个数字有振动频率（vibration）。

**和东方八字的对比：**

| | 西方数术 | 东方八字 |
|---|---------|----------|
| 核心 | 数字（1-9） | 天干地支 |
| 方法 | 加法化简 | 排盘 |
| 用途 | 性格分析 | 命运趋势 |
| 上手难度 | 极低 | 中 |

## 三个核心数字

### 1. 生命灵数 = 你的性格核心

**算法：** 出生年月日所有数字相加，化简到1-9。

**例外：** 如果得到11/22/33（Master Numbers），不化简。

```
例子：1995年3月21日
1+9+9+5 + 3 + 2+1 = 30
3+0 = 3 → 生命灵数3
```

### 2. 命运数 = 你的天赋和社会角色

**算法：** 名字里每个字母对应数字（A=1, B=2... I=9, J=1...），全部加起来化简。

### 3. 灵魂数 = 你内心的渴望

**算法：** 只用名字里的**元音字母**（A/E/I/O/U）对应的数字，加起来化简。

## 学习经验

### 入门（1天）

**目标：** 学会算自己/朋友的生命灵数。

**方法：**
1. 算自己的：出生日期全加 → 化简 → 得灵数
2. 看灵数含义
3. 对比自己的性格——准吗？

### 进阶（1周）

**目标：** 学命运数和灵魂数。

**方法：**
1. 把自己的名字转成数字
2. 命运数（全部字母）= 你在社会上的角色
3. 灵魂数（只用元音）= 你内心的渴望
4. 对比三个数字——命运数=别人看到的你；灵魂数=你自己知道的你

### 精通（2周+）

**目标：** 学个人年（每9年一个循环），学名字和出生日期的更深对应。
