# 西方数术

> 数字本身有意义，用数字来解读性格和趋势。

---

## 什么是西方数术

西方数术（Numerology）是通过数字来分析性格和预测趋势的体系。

**核心信念：**
- 每个数字有独特的振动频率（vibration）
- 名字和生日里的数字，揭示了你的性格和命运

---

## 基础概念

### 生命灵数（Life Path Number）

用生日算出，代表你的人生主题。

**计算方法：**
生日：1985年7月23日
```
1+9+8+5 + 7 + 2+3 = 35
3+5 = 8
生命灵数 = 8
```

> 💡 **注意**：如果中间结果是11、22、33，不要化简（它们是"主数"，有特殊意义）。
