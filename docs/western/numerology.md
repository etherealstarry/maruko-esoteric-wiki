## 什么是西方数术

西方数术（Numerology）是用**出生日期**和**名字**来推算性格和命运趋势的体系。

**核心想法：** 数字不是随机的，每个数字都有振动频率（vibration），影响着人的性格和命运。

---

## 生命灵数（Life Path Number）

最重要的一个数，用**出生日期**算出。

### 计算方法

把出生日期的年月日所有数字加起来，化简到1-9。

**例子：1995年3月21日**
```
1+9+9+5 + 3 + 2+1 = 30
3+0 = 3
生命灵数 = 3
```
